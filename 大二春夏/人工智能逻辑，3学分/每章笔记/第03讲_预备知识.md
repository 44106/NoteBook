## 3.1 集合论初步

集合论是现代数学的基础。**集合**由某些对象汇集而成。称这些对象为集合的**元**。用 `a ∈ S` 表示 `a` 是集合 `S` 的元。用 `a ∉ S` 表示 `a` 不是集合 `S` 的元。

集合通常用大写的英文字母（如 `A, B, C` 等）表示，而集合中的元素则用小写的英文字母或数字表示，并用花括号"`{ }`"括起来。例如，集合 `A = {1, 2, 3}` 包含三个元素：1、2 和 3。

集合所包含元的全体称为它的**外延**。集合的**内涵**是它的元所共有的性质。

### 子集与幂集

令 `S` 和 `T` 为两个集合：

- 如果对于所有 `x`，有 `x ∈ S` 蕴涵 `x ∈ T`，那么称 `S` 为 `T` 的**子集**，记作 `S ⊆ T`。
- 如果 `S ⊆ T` 且 `S ≠ T`，那么称 `S` 为 `T` 的**真子集**，记作 `S ⊂ T`。

注意：所有集合都是自己的子集，但**不是**自己的真子集。

称一个集合的所有子集的集合为该集合的**幂集**。

### 空集

**空集** `∅` 是一个特殊的集合，其中没有元。`∅` 是任何集合的子集。

### 极大、极小、最大、最小

给定一个**集合族**（由集合构成的集合）`S`，对于任意 `S ∈ S`：

- 如果不存在 `T ∈ S` 使得 `S ⊂ T`，那么称 `S` 在 `S` 中是**极大的**；
- 如果不存在 `T ∈ S` 使得 `T ⊂ S`，那么称 `S` 在 `S` 中是**极小的**；
- 如果对于所有 `T ∈ S`，都有 `T ⊆ S`，那么称 `S` 在 `S` 中是**最大的**；
- 如果对于所有 `T ∈ S`，都有 `S ⊆ T`，那么称 `S` 在 `S` 中是**最小的**。

可以看出，最大的集合要么不存在，要么存在且唯一，最小的集合同理。另外：

- 最大的集合一定是极大的
- 最小的集合一定是极小的
- 但**极大的集合未必是最大的，极小的集合也未必是最小的**

### 集合的表示法

用 `{x | ---x---}` 表示由所有使得 `---x---`（它是一个讲到 `x` 的命题）成立的对象 `x` 构成的集合。

### 集合的基本操作

集合的基本操作包括**补、并、交和差**：

- **补集**：`S̄ = {x | x ∉ S}`（即 `S` 的补集由所有不属于 `S` 的元素构成）
- **并集**：`S ∪ T = {x | x ∈ S 或 x ∈ T}`
- **交集**：`S ∩ T = {x | x ∈ S 并且 x ∈ T}`
- **差集**：`S - T = {x | x ∈ S 并且 x ∉ T}`（原文此处有笔误，标准定义为属于S但不属于T的元素）

此外，若 `S ∩ T = ∅`，我们称 `S` 与 `T` **不相交**。

### 有序偶与笛卡尔积

**有序偶**是由两个元素按照一定顺序组成的组合，强调元素的顺序性。由 `a` 和 `b` 组成的有序偶记作 `(a, b)`。

- 若两个有序偶的元素相同且顺序一致，则它们相等，即 `(a, b) = (c, d)` 当且仅当 `a = c` 并且 `b = d`。
- 如果元素相同但顺序不同，则视为不同的有序偶，比如 `(1, 2)` 和 `(2, 1)` 是不同的有序偶。

**有序n元组** `(a_1, ..., a_n)` 与有限序列 `a_1, ..., a_n` 相同。有序n元组构成的集合也记作：

```
{(x_1, ..., x_n) | ---x_1---, 并且 ..., 并且 ---x_n---}
```

**笛卡尔积**：集合 `S_1, ..., S_n` 的笛卡尔积 `S_1 × ... × S_n` 的定义如下：

```
S_1 × ... × S_n = {(x_1, ..., x_n) | x_1 ∈ S_1, 并且 ..., 并且 x_n ∈ S_n}
```

设 `S` 是一个集合。当 `S_1 = ... = S_n = S` 时，`S` 的 `n` 次笛卡尔积记作 `S^n`，即：

```
S^n = S × ... × S（共n个S）
```

表示所有n元组 `(x_1, ..., x_n)` 组成的集合，其中每个 `x_i ∈ S`。

在笛卡尔积这个概念的基础上，可以定义**关系**和**函数**。

### 关系

当 `n > 2` 时，集合 `S` 上的**n元关系** `R` 定义如下：

```
R = {(x_1, ..., x_n) | x_1, ..., x_n ∈ S，并且 x_1, ..., x_n 之间（依次）有R关系}
```

因此，`R ⊆ S^n`。

当 `n = 1` 时，`S` 上的**一元关系** `R` 是 `S` 元中的一个**性质**：

```
R = {x ∈ S | x 有R性质}
```

这时有 `R ⊆ S`。

### 关系的主要性质

**定义3.1（关系的特性）**：给定集合 `X`，对于任意 `a, b, c ∈ X`，`X` 上的**二元关系** `R ⊆ X × X` 满足：

- **自反性**：`(a, a) ∈ R`
- **对称性**：`(a, b) ∈ R` 当且仅当 `(b, a) ∈ R`
- **传递性**：如果 `(a, b) ∈ R` 且 `(b, c) ∈ R`，则 `(a, c) ∈ R`

**定义3.2（关系的传递闭包）**：给定集合 `X` 上的二元关系 `R ⊆ X × X`，`R` 的**传递闭包** `R^+` 是 `X` 上最小的包含 `R` 的传递关系，即满足以下条件的最小关系：

1. **包含R**：`R ⊆ R^+`
2. **传递性**：对于任意 `a, b, c ∈ X`，如果 `(a, b) ∈ R^+` 且 `(b, c) ∈ R^+`，则 `(a, c) ∈ R^+`
3. **最小性**：如果 `S` 是任何满足条件 (1) 和 (2) 的关系，则 `R^+ ⊆ S`

**定义3.3（等价关系）**：给定集合 `X` 上的二元关系 `R ⊆ X × X` 是一个**等价关系**，当且仅当它满足**自反性、对称性和传递性**。

关于 `R` 下 `a` 的**等价类**，记作 `[a]`，定义为：

```
[a] = {x ∈ X | (x, a) ∈ R}
```

**例题3.1（等价关系-等价类-划分）**：在集合 `X = {a, b, c}` 上，关系 `R = {(a, a), (b, b), (c, c), (b, c), (c, b)}` 是等价关系。

- 关于R的等价类：`[a] = {a}`，`[b] = [c] = {b, c}`
- 集合X中关于R的所有等价类的集合为 `{{a}, {b, c}}`

我们说 `{{a}, {b, c}}` 是集合 `X` 关于 `R` 的一个**划分**。在这里，说 `{X_1, ..., X_n}` 是 `X` 的一个划分，当且仅当：

1. `X_i ≠ ∅`（每个子集非空）
2. `X_i ∩ X_j = ∅`（其中 `i, j = 1, ..., n, i ≠ j`，任意两个不同的子集不相交）
3. `X_1 ∪ ... ∪ X_n = X`（所有子集的并集等于原集合）（原文此处使用了∩，应为∪）

### 函数

**一元函数** `f` 是由序偶构成的集合，使得如果 `(x, y) ∈ f`，并且 `(x, z) ∈ f`，则 `y = z`。

`f` 的**定义域** `dom(f)` 是集合：

```
dom(f) = {x | 存在y使得 (x, y) ∈ f}
```

`f` 的**值域** `ran(f)` 是集合：

```
ran(f) = {y | 存在x使得 (x, y) ∈ f}
```

若 `f` 是函数，并且 `dom(f) = S`，`ran(f) ⊆ T`，则称 `f` 为**由S到T（中）的函数**，记作：

```
f: S → T
```

若除此之外还有 `ran(f) = T`，则称 `f` 映射 `S` 到 `T` 上（`f` 是**满射**）。

集合 `S` 的**n元函数** `f` 是映射 `S^n` 到 `S` 的函数。

### 函数的不动点

函数的**不动点**是一个重要的概念，它描述了函数与自身输入输出之间的一种特殊"稳定"关系。直观地说，如果一个点经过某函数映射后仍然回到自身，这个点就是该函数的不动点。

形式化定义：令 `f` 是一个从 `S` 到 `S` 的函数（`f: S → S`），若存在某个元素 `x ∈ S`，使得 `f(x) = x`，则称 `x` 是函数 `f` 的一个**不动点**。

比如，考虑 `f_1(x) = x^3` 这个从实数集到实数集的函数，`x = 1` 是 `f_1` 的一个不动点，因为 `f_1(1) = 1`；而 `x = 2` 不是 `f_1` 的不动点，因为 `f_1(2) ≠ 2`。

## 3.2 图论初步

图论是研究由点和边构成的结构的学科。

### 图的基本定义

**图**是一个二元组 `G = (V, E)`，其中：

- `V` 是**顶点**（也称为**节点**）的集合
- `E ⊆ V × V` 是**边**的集合

如果对于任意两个不同的顶点 `u, v ∈ V`，都有 `(u, v) ∈ E`，则 `G` 为**完全图**。

### 有向边与有向图

当边 `e = (v_i, v_j)` 为顶点的有序对时，称其为**有向边**，其中 `v_i` 被称作这条边的**起点**，`v_j` 被称为**终点**。

- 如果图中的所有边都是有向边，称图为**有向图**。
- 如果一个图的所有边都没有方向性，则称之为**无向图**。

### 子图

给定图 `G_1 = (V_1, E_1)` 和 `G_2 = (V_2, E_2)`：

- 如果 `V_1 ⊆ V_2` 且 `E_1 ⊆ E_2`，称 `G_1` 是 `G_2` 的**子图**，记为 `G_1 ⊆ G_2`。
- 如果 `G_1 ⊆ G_2` 且 `G_1 ≠ G_2`，称 `G_1` 是 `G_2` 的**真子图**。
- 如果 `V_1 = V_2` 且 `E_1 ⊆ E_2`，称 `G_1` 是 `G_2` 的**生成子图**。
- 如果 `V_1 ⊆ V_2` 且 `E_1 = E_2 ∩ (V_1 × V_1)`，称 `G_1` 是 `G_2` 的**导出子图**。

### 途径、迹、路、圈

给定图 `G = (V, E)`，序列 `v_0 e_1 v_1 e_2 ... e_n v_n` 称为从顶点 `v_0` 到顶点 `v_n` 的**途径**，其中 `v_i ∈ V`，`e_i = (v_{i-1}, v_i) ∈ E`，`1 ≤ i ≤ n`。

- **闭途径**：如果 `v_0 = v_n`，称该途径为闭途径。
- **迹**：如果一条途径中的所有边均不相同，称该途径为迹。
- **路**：称一条迹为路，如果它的任意两个顶点互异。
- **圈（环）**：一条闭迹为圈（环），如果除了第一个顶点和最后一个顶点，其他顶点均互异。

### 可达性与连通性

在图 `G = (V, E)` 中：

- 如果顶点 `v_1` 到顶点 `v_2` 有路，称 `v_1` 到 `v_2` 是**可达**的，否则称 `v_1` 到 `v_2` **不可达**。
- 如果对于任意 `v_i, v_j ∈ V`，`v_i` 到 `v_j` 是可达的，`v_j` 到 `v_i` 也是可达的，称该图为**强连通图**。
- 没有有向圈的有向图称为**有向无环图（DAG）**。

### 强连通分量

给定图 `G = (V, E)`，`G_1 ⊆ G`，如果 `G_1` 是强连通的，且不存在 `G_2` 使得 `G_1 ⊂ G_2` 且 `G_2` 是强连通的，那么称 `G_1` 是 `G` 的**强连通分量（SCC）**。

**例题3.2（有向图）**：图3.1是一个包含两个顶点的有向图。由该图的所有强连通分量构成的缩减图是一个有向无环图。因此，基于强连通分量的概念，可以把一个有向图分解为若干个子图。

**例题3.3（强连通分量分解）**：图3.2中的有向图可以被分解为四个强连通分量（SCC）。图中用蓝色区域标识了这四个分量。

### 图的合并

考虑两个图 `G_1 = (V_1, E_1)` 和 `G_2 = (V_2, E_2)`。`G_1` 和 `G_2` 的**合并**，记作 `G_1 ∪ G_2`，是一个图，其顶点集是 `V_1 ∪ V_2`，边集是 `E_1 ∪ E_2`。

### 树与有根树

一个无向图若连通且不含圈，则称其为一棵**树**。在一棵树中，任意两个节点均由唯一的路连接。

当指定树中某一个节点为根时，称为**有根树**，此时可以定义层次结构：

- **根节点**：有根树中指定的"最顶层节点"，是树中唯一没有父节点的节点，根节点是所有其他节点的祖先。
- **父节点与子节点**：对于任意两个不同的节点 `u, v ∈ V`，若 `u` 和 `v` 间有一条边，且 `u` 是 `v` 到根节点路径上的前一个节点，则称 `u` 是 `v` 的父节点，称 `v` 是 `u` 的子节点。在一棵有根树中，每个节点（除根节点外）有且仅有一个父节点，一个父节点可以有多个子节点。
- **祖先节点与后代节点**：对于节点 `u ∈ V`，从根节点到 `u` 的路上的所有节点（包括父节点、祖父节点等）均为 `u` 的祖先节点。注意，节点自身不是其祖先节点。所有以 `u` 为祖先节点的节点都是 `u` 的后代节点。
- **叶子节点**：没有子节点的节点称为叶子节点，它们位于树的"末端"。

**图3.3（有根树示例）**：图3.3是一棵有根树 `G_3`，其中：

- 根节点是 `α_1`
- 叶子节点是 `α_3`、`α_5`、`α_7`、`α_8`、`α_9`
- `α_1` 的子节点是 `α_2`、`α_3`、`α_4`
- `α_6` 的父节点是 `α_2`
- `α_8` 的祖先节点是 `α_1`、`α_2`、`α_6`
- `α_8` 的后代节点是 `α_9`

## 3.3 小结

本讲作为形式逻辑系统的数学预备，重点介绍了**集合论**与**图论**的基本概念。

**集合、关系与函数**构成了逻辑语言语法与语义的基本载体：

- **集合**用于界定命题所涉及对象的范围
- **关系**用于描述对象之间的结构性联结
- **函数**用于刻画对象间的映射关系

这些概念不仅在命题逻辑与一阶逻辑中频繁出现，也在描述逻辑、缺省推理等后续系统中持续发挥基础作用。

另一方面，**图论**为人工智能逻辑中的结构建模提供了关键工具。尤其在构建知识图谱、分析贝叶斯网络、表达因果关系与论证结构时，图模型扮演着核心角色。**有向图**与**有向无环图**能够形式化表示因果路径、推理链路与知识依赖结构，是理解复杂推理系统不可或缺的前提。

通过本讲学习，读者不仅掌握了后续章节中逻辑语言与系统建模所需的基本数学工具，也初步建立起对人工智能逻辑建模中**"结构—语义—推理"**三者关系的理解：

- **集合与图**为逻辑提供结构
- **语义解释**使这些结构与现实世界对应
- **推理机制**则在此基础上实现知识的动态演绎与验证

下一讲将正式进入逻辑系统的构造与推理方法，从命题逻辑出发，探索如何用形式语言精确定义知识、进行可计算的推理，并逐步走向面向智能任务的复杂逻辑系统。

## 学习要点与AI关联

- **集合论**：集合、关系与函数是逻辑系统的基本对象。集合运算为数据库查询、知识图谱存储、逻辑语言建模提供形式基础。关系与函数则常用于一阶逻辑建模、知识表示与推理。
- **图论**：图结构为人工智能逻辑中的结构建模提供关键工具。有向图和有向无环图可形式化表示因果路径、推理链路与知识依赖，广泛应用于贝叶斯网络、知识图谱、论辩框架建模等任务。
- **逻辑联系**：集合论与图论共同奠定了"结构—语义—推理"三位一体的基础。集合论提供了对象与关系的抽象结构，图论揭示了复杂系统中的联结与依赖关系；二者结合为符号模型提供了语义解释的支撑，使形式结构能够对应现实世界中的实体与关系；在此基础上，推理机制才能发挥作用，从而形成后续形式逻辑与推理系统的理论准备。
- **AI应用价值**：本讲所述数学工具服务于知识表示、推理算法设计、因果建模、语义搜索等领域，使智能系统具备结构化建模与高效计算的能力。
- **哲学与方法论启示**：集合论体现了对象与整体的关系，图论揭示了结构与联系的形式。它们不仅为人工智能提供了建模工具，也为哲学中关于"个体—整体""部分—关系"的经典问题提供了形式化分析路径。

## 第3讲练习

### 单选题

1. 以下（ **C** ）是集合 `{1, 2, 3}` 的幂集。
   - A. `{{1}, {2}, {3}}`
   - B. `{1, 2, 3}`
   - C. `{∅, {1}, {2}, {3}, {1,2}, {2,3}, {1,3}, {1,2,3}}`
   - D. `{∅, {1}, {2}, {3}, {1,2}, {1,3}, {2,3}}`

   **解析**：幂集是集合所有子集的集合。`{1,2,3}`的子集有：`∅`、`{1}`、`{2}`、`{3}`、`{1,2}`、`{1,3}`、`{2,3}`、`{1,2,3}`，共8个。选项D遗漏了空集和全集。

2. 设 `A = {1, 2, 3}`，`B = {a, b}`，则 `A × B` 等于（ **A** ）。
   - A. `{(1, a), (1, b), (2, a), (2, b), (3, a), (3, b)}`
   - B. `{(a, 1), (a, 2), (a, 3), (b, 1), (b, 2), (b, 3)}`
   - C. `{1, 2, 3, a, b}`
   - D. `∅`

   **解析**：笛卡尔积 `A × B` 是A中每个元素与B中每个元素配对，且顺序为A的元素在前。

3. 关系 `R = {(1,1), (2,2), (3,3), (1,2), (2,1)}` 在集合 `{1,2,3}` 上是（ **A** ）。
   - A. 自反的，对称的，但不是传递的
   - B. 自反的，传递的，但不是对称的
   - C. 对称的，传递的，但不是自反的
   - D. 自反的，对称的，传递的

   **解析**：`(1,1), (2,2), (3,3)` 都在R中，所以自反；`(1,2)` 和 `(2,1)` 都在R中，所以对称；但 `(2,1)` 和 `(1,2)` 在R中，却没有推出 `(2,2)` 以外的结果，尤其是 `(1,1)` 和 `(2,2)` 并不能满足传递性需要——需要检查 `(x,y)` 和 `(y,z)` 是否蕴含 `(x,z)`。由于有 `(1,2)` 和 `(2,1)` 但没有其他传递要求违反，实际上该关系的传递闭包包含了 `(1,1),(2,2),(3,3),(1,2),(2,1)`。

4. 函数 `f: S → T` 必须满足（ **A** ）。
   - A. S中的每个元素都映射到T中的一个元素。
   - B. T中的每个元素都映射到S中的一个元素。
   - C. S中的每个元素都映射到T中的多个元素。
   - D. S和T中的元素一一对应。

   **解析**：函数的定义要求定义域S中的每个元素必须映射到值域T中的**恰好一个**元素（唯一性）。

5. 以下哪种关系在集合X上是等价关系？（ **C** ）
   - A. 小于关系 (`<`)
   - B. 小于等于关系 (`≤`)
   - C. 等于关系 (`=`)
   - D. 不等于关系 (`≠`)

   **解析**：等于关系满足自反性（`a = a`）、对称性（`a = b` 当且仅当 `b = a`）、传递性（`a = b` 且 `b = c` 则 `a = c`），因此是等价关系。小于关系不满足自反性和对称性；小于等于关系不满足对称性；不等于关系不满足自反性。

6. 一个有向图中，如果从任意一个顶点到另一个顶点都存在一条路径，那么该图是（ **B** ）。
   - A. 连通图
   - B. 强连通图
   - C. 完全图
   - D. 树

   **解析**：强连通图的定义要求任意一对顶点 `u, v`，从 `u` 到 `v` 和从 `v` 到 `u` 都是可达的。选项A（连通图）是对于无向图的概念。

7. 一条路是指（ **C** ）。
   - A. 顶点和边交替出现的序列
   - B. 顶点和边交替出现的序列，且所有边都不同
   - C. 顶点和边交替出现的序列，且所有顶点都不同
   - D. 顶点和边交替出现的序列，且所有顶点和边都不同

   **解析**：路的定义要求所有顶点互异（除起点和终点可能相同外）。选项B是"迹"的定义。

8. 以下（ **D** ）一定是有向无环图。
   - A. 强连通图
   - B. 完全图
   - C. 树
   - D. 由一个有向图的所有强连通分量构成的缩减图

   **解析**：强连通分量的缩减图（将每个SCC缩成一个节点）必然是有向无环图（DAG），因为如果存在环，那么环上的SCC将会合并成一个更大的SCC。

9. 设G是一个有向图，G的强连通分量是指（ **A** ）。
   - A. G的极大强连通子图
   - B. G的任意一个强连通子图
   - C. G的所有强连通子图的并集
   - D. G的最大强连通子图的集合

   **解析**：强连通分量（SCC）的定义是：一个强连通子图，且不能被任何更大的强连通子图包含（即极大的）。

10. 集合 `{a, b, c}` 上的二元关系 `R = {(a, a), (b, b), (c, c)}` 是（ **A** ）。
    - A. 自反的，对称的，传递的
    - B. 自反的，但不一定是对称的或传递的
    - C. 对称的，但不一定是自反的或传递的
    - D. 传递的，但不一定是自反的或对称的

    **解析**：这是恒等关系（identity relation）。每个元素都与自己相关（自反）；如果 `(x, y) ∈ R` 则必有 `x = y`，所以 `(y, x) ∈ R`（对称）；如果有 `(x, y) ∈ R` 且 `(y, z) ∈ R`，则必有 `x = y = z`，所以 `(x, z) ∈ R`（传递）。因此它是等价关系。

### 简答题

11. **设R是集合A上的一个等价关系。证明：R的传递闭包也是一个等价关系。**

    答：由于R是等价关系，R本身满足自反性、对称性和传递性。R的传递闭包 `R^+` 是包含R的最小传递关系。由于R本身已经是传递的，所以 `R^+ = R`。因此 `R^+` 等同于R，自然也满足自反性、对称性和传递性，即是等价关系。

12. **设R和S是集合A上的两个等价关系。证明：R ∩ S 也是A上的一个等价关系。**

    答：需要证明 `R ∩ S` 满足自反性、对称性和传递性。
    - **自反性**：因为R和S都是自反的，所以对任意 `a ∈ A`，有 `(a, a) ∈ R` 且 `(a, a) ∈ S`，因此 `(a, a) ∈ R ∩ S`。满足自反性。
    - **对称性**：若 `(a, b) ∈ R ∩ S`，则 `(a, b) ∈ R` 且 `(a, b) ∈ S`。因为R和S都是对称的，所以 `(b, a) ∈ R` 且 `(b, a) ∈ S`，因此 `(b, a) ∈ R ∩ S`。满足对称性。
    - **传递性**：若 `(a, b) ∈ R ∩ S` 且 `(b, c) ∈ R ∩ S`，则分别有 `(a, b), (b, c) ∈ R` 和 `(a, b), (b, c) ∈ S`。因为R和S都是传递的，所以 `(a, c) ∈ R` 且 `(a, c) ∈ S`，因此 `(a, c) ∈ R ∩ S`。满足传递性。
    因此，`R ∩ S` 是一个等价关系。

13. **一个无向图G有n个顶点和m条边。如果G是一棵树，那么m和n之间的关系是什么？**

    答：如果G是一棵树，则它是连通的且不含圈。根据树的性质，m和n之间的关系为：
    ```
    m = n - 1
    ```
    即边的数量等于顶点数量减1。这是因为树是极小连通图，每增加一个顶点恰好需要增加一条边来保持连通，从1个顶点（0条边）开始，逐步增加。

14. **如果10个人中每个人都与其他人握手，总共发生多少次握手？这个问题与图论有什么关系？**

    答：总共有 `C(10, 2) = 10 × 9 / 2 = 45` 次握手。

    这个问题对应图论中的**完全图**：将每个人看作一个顶点，每次握手看作一条连接两个顶点的边。每个人都与其他人握手，意味着任意两个不同的顶点之间都有一条边。因此，该图是一个10个顶点的完全图 `K_10`。完全图的边数为 `n(n-1)/2`，代入 `n = 10` 得到45条边，即45次握手。

15. **令 `A = {1, 2, 3, 4, 5, 6}`，`B = {1, 2, 3, 4}`，`R ⊆ A × B`。已知对任意 `a ∈ A`，任意 `b ∈ B`，有 `(a, b) ∈ R` 当且仅当 `(a - b) mod 2 = 0`，其中 `(a - b) mod 2` 是一个模运算，它表示 `a - b` 除以2的余数。请写出R的所有元素。**

    答：条件 `(a - b) mod 2 = 0` 等价于 `a` 和 `b` 具有相同的奇偶性（同奇或同偶）。

    A = {1, 2, 3, 4, 5, 6}：
    - 奇数：1, 3, 5
    - 偶数：2, 4, 6

    B = {1, 2, 3, 4}：
    - 奇数：1, 3
    - 偶数：2, 4

    R = {(a, b) | a与b同奇偶}
    = {(1,1), (1,3), (3,1), (3,3), (5,1), (5,3)} ∪ {(2,2), (2,4), (4,2), (4,4), (6,2), (6,4)}
    = {(1,1), (1,3), (2,2), (2,4), (3,1), (3,3), (4,2), (4,4), (5,1), (5,3), (6,2), (6,4)}

16. **建立一个有向图来表示在集合 `{1, 2, 4, 5, 10, 20}` 上定义的关系S，使得 `(x, y) ∈ S` 当且仅当 `x < y` 且 `x` 整除 `y`。**

    答：需要检查每对可能的 `(x, y)`：
    - 1整除所有数（1 < 2, 4, 5, 10, 20）：(1,2), (1,4), (1,5), (1,10), (1,20)
    - 2整除4, 10, 20（2 < 4, 10, 20，且2 | 4, 2 | 10, 2 | 20）：(2,4), (2,10), (2,20)
    - 4整除20（4 < 20）：(4,20)
    - 5整除10, 20（5 < 10, 20，且5 | 10, 5 | 20）：(5,10), (5,20)
    - 10整除20（10 < 20，且10 | 20）：(10,20)
    - 2不整除5（不满足整除条件）
    - 4不整除5, 10（不满足整除条件）
    - 5不整除4（不满足整除条件）

    有向图 `G = (V, E)`：
    - V = {1, 2, 4, 5, 10, 20}
    - E = {(1,2), (1,4), (1,5), (1,10), (1,20), (2,4), (2,10), (2,20), (4,20), (5,10), (5,20), (10,20)}

    这个图是一个**有向无环图（DAG）**，可以用哈斯图的方式分层排列：1在底层，2和5在第二层，4和10在第三层，20在顶层。
