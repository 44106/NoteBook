# 第5讲 一阶逻辑

## 引言

上一讲介绍的命题逻辑语言只涉及命题和联结词，而没有考虑到命题内部的各个成分以及基于这些成分的逻辑结构。为了表达更加复杂的逻辑关系，在知识的表示与推理时，经常要用到**谓词**、**量词**、**对象**、**论域**等概念。其中，谓词用于表达一个语句所谈论的对象的属性或对象之间的关系，而量词用于指定谓词所涉及的对象的范围。一阶逻辑研究包含谓词和量词的语句的语法构成、真值条件、逻辑推理和形式推演。因此，与命题逻辑相比，**一阶逻辑具有更强的表达能力**。

在一阶逻辑的框架下研究知识的表示和推理，需要深入了解如下几个方面的内容：谓词和量词的概念、一阶语言、对一阶语言的解释、一阶语义蕴涵、形式推演等。

---

## 5.1 谓词和量词

命题逻辑仅通过真值联结词来刻画命题之上的逻辑关系，但未涉及简单命题内部结构，因此无法刻画基于命题内部结构的逻辑关系。

### 例5.1（有关张三的妈妈的命题）

> 每位航天员都接受过严苛的训练。张三的妈妈是航天员。所以，张三的妈妈接受过严苛的训练。

为了表示上述例子，需要引入**对象**、**论域**、**函词**、**谓词**和**量词**等概念。

### 基本概念

抽象地看，世界由对象及其关系构成。

**论域**：把所有被讨论对象的集合称为论域。论域中的元素，即被讨论的对象，称为**个体**。个体可以由常元或变元来描述。

**常元**：用于表示确定对象的符号。例如，可以用"张三"表示一个确定的个体。

**变元**：用于表示给定论域上的任意一个对象。例如，给定全总个体域，语句"$$x$$ 是航天员"中，$$x$$ 是变元。

**函词**：给定一个论域，从一组个体到单个个体的映射关系可以用函词来描述。例如，在"张三的妈妈"中，"......的妈妈"是一个函词。

**谓词**：个体之间的关系用谓词来描述。每个谓词附带着可以放置所讨论对象的位置，称为**空位**。把谓词携带空位的数目称为谓词的**元数**。用一元谓词表示的关系也称为**个体的性质**。

把谓词、常元、函词等结合在一起可以表示一些命题。"张三的妈妈是航天员"是一个命题。其中：
- "张三"是常元
- "......的妈妈"是函词
- "......是航天员"是谓词

### 量词

除了单称命题，在自然语言中，还包括全称命题和特称命题。为了表达这类命题需要引入**量词**的概念。量词包括两类：

- **全称量词**：用于描述某个变元的取值范围涵盖一整个论域，记作 $$\forall x$$，读作"对于所有的 $$x$$"。
- **存在量词**：用于描述在给定论域中存在某个个体，记作 $$\exists x$$，读作"存在 $$x$$"。

### 核心概念理解：论域与量词

- **论域**：就是我们讨论问题时涉及的"对象池"。例如：论域 = 所有学生。
- **全称量词** $$\forall x$$：表示"对论域里的每个对象 $$x$$"。
- **存在量词** $$\exists x$$：表示"至少有一个对象 $$x$$"。

通过上述分析可知，为了表示自然语言语句的内部结构，涉及谓词、量词、个体常元、个体变元、函词和联结词等概念的使用。其中，联结词与命题逻辑中的一样，而**谓词和量词是一阶逻辑的主要研究对象**。

## 5.2 一阶语言

一阶语言 $$\mathcal{L}$$ 含七类符号。

### 七类符号

**第一类：个体符号**，通常表示为正体小写拉丁文字母（或加其他记号）：$$a, b, c$$

**第二类：个体变元符号**，包括自由变元符号和约束变元符号。
- 约束变元符号通常表示为：$$x, y, z$$
- 自由变元符号通常表示为：$$u, v, w$$

**第三类：函数符号**，对应于自然语言语句中的函词，通常表示为正体小写拉丁文字母（或加其他记号）：$$f, g, h$$
任何函数符号 $$f$$ 有确定的元数 $$m \geq 1$$，称元数为 $$m$$ 的 $$f$$ 为 $$m$$ 元函数符号。

**第四类：关系符号**，对应于自然语言语句中的谓词，通常用正体大写拉丁文字母（或加其他记号）：$$F, G, H$$
任何关系符号 $$F$$ 有确定的元数 $$n \geq 1$$，称元数为 $$n$$ 的 $$F$$ 为 $$n$$ 元关系符号。等于符号 $$\equiv$$ 是一种特殊的关系符号。

**第五类：量词符号** $$\forall$$ 和 $$\exists$$。

**第六类：联结符号**，与命题逻辑的相同：$$\neg, \land, \lor, \to, \leftrightarrow$$。

**第七类：标点符号**，包括左括号、右括号和逗号。

### 项和公式的定义

### 定义5.1（项）

项可以定义如下：

1. 变元和常元是项。
2. 如果 $$t_1, \ldots, t_m$$ 是项，$$f$$ 是 $$m$$ 元函数符号，则 $$f(t_1, \ldots, t_m)$$ 是项。
3. 只有经过有限次使用(1)(2)条款生成的符号串才是项。

不含自由变元的项称为**基项**。在例5.1中，"张三"用个体符号 $$ZS$$ 表示，"张三的妈妈"用 $$g(ZS)$$ 表示。这时，$$ZS$$ 和 $$g(ZS)$$ 都是基项，表示的是论域中确定的个体。

### 定义5.2（公式）

公式可以定义如下：

1. $$F(t_1, \ldots, t_n)$$ 是公式，称为**原子公式**。其中，$$F$$ 是 $$n$$ 元关系符号，且 $$t_1, \ldots, t_n$$ 是项。
2. 如果 $$t_1$$ 和 $$t_2$$ 是项，那么 $$(t_1 \equiv t_2)$$ 是原子公式。
3. 如果 $$\phi$$ 和 $$\psi$$ 是公式，则 $$(\neg\phi), (\phi \land \psi), (\phi \lor \psi), (\phi \to \psi), (\phi \leftrightarrow \psi)$$ 是公式。
4. 如果 $$\phi(u)$$ 是公式，并且 $$x$$ 不在 $$\phi(u)$$ 中出现，则 $$\forall x\phi(x), \exists x\phi(x)$$ 是公式。
5. 只有经过有限次使用(1)(2)(3)(4)条款生成的符号串才是公式。

### 量词的辖域与变元的约束

紧跟在量词后面的公式被称为量词的**辖域**。在量词及其辖域中的变元出现被称为变元的**约束出现**，否则称为**自由出现**。约束出现的变元被称为**约束变元**，自由出现的变元被称为**自由变元**。不含有自由变元的公式被称为**闭公式**或**句子**。

一个公式中的自由变元可以被项代换，代换后，变为一个新的公式。

### 定义5.3（代换）

代换 $$\theta$$ 是一个有限的对子集合 $$\{u_1/t_1, \ldots, u_n/t_n\}$$，其中 $$u_i$$ 是自由变元，$$t_i$$ 是项。如果 $$\phi$$ 是一个公式，$$\theta$$ 是一个代换，则 $$\phi\theta$$ 是一个公式。在该公式中，所有 $$u_i$$ 的自由出现都被替换为 $$t_i$$。有时，我们也把 $$\phi\{u/t\}$$ 写作 $$\phi^u_t$$。

## 5.3 自然语言的一阶表示

一阶语言可以对各种不同类型的自然语言语句进行符号化。

### 一、四种直言命题

首先，考虑如下四种直言命题：

1. **全称肯定命题**："所有 $$S$$ 都是 $$P$$"
2. **全称否定命题**："所有 $$S$$ 都不是 $$P$$"
3. **特称肯定命题**："有的 $$S$$ 是 $$P$$"
4. **特称否定命题**："有的 $$S$$ 不是 $$P$$"

通常有两种方式来符号化直言命题：

**方式一**：当假设论域为 $$S$$ 时：
1. $$\forall x P(x)$$
2. $$\forall x \neg P(x)$$
3. $$\exists x P(x)$$
4. $$\exists x \neg P(x)$$

**方式二**：当假设论域为全体个体时：
1. $$\forall x (S(x) \to P(x))$$
2. $$\forall x (S(x) \to \neg P(x))$$
3. $$\exists x (S(x) \land P(x))$$
4. $$\exists x (S(x) \land \neg P(x))$$

以第二种形式化方式为例，"所有的鱼都生活在水里"可以被形式化为 $$\forall x (F(x) \to G(x))$$，其中 $$F$$ 表示"......是鱼"，$$G$$ 表示"......是生活在水里"。

**重要提示**：全称命题使用蕴涵（$$\to$$），特称命题使用合取（$$\land$$）。这是因为：
- "所有 $$S$$ 都是 $$P$$" 意味着：对任意对象，如果它是 $$S$$，那么它是 $$P$$。
- "有的 $$S$$ 是 $$P$$" 意味着：存在对象，它既是 $$S$$ 又是 $$P$$。

### 二、一般的关系命题

考虑："所有孤竹国公民都认识所有的采薇者"。该命题可以被符号化为：

$$\forall x (F(x) \to \forall y (G(y) \to H(x, y)))$$

其中：
- $$F$$ 表示"......是孤竹国公民"
- $$G$$ 表示"......是采薇者"
- $$H$$ 表示"......认识......"

### 三、否定式的量化

"一切事物都不是尽善尽美的"可以被形式化为 $$\forall x \neg F(x)$$，其中 $$F$$ 表示"......是尽善尽美的"。

"所有非正义战争都是不得人心的"也是一个否定式的量化。它可以被形式化为 $$\forall x (\neg F(x) \to \neg G(x))$$，其中：
- $$F$$ 表示"......是正义战争"
- $$G$$ 表示"......是得人心的"

### 四、量化式的否定

"并非有人是天生的科学家"可以被形式化为 $$\neg \exists x (F(x) \land G(x))$$，其中：
- $$F$$ 表示"......是人"
- $$G$$ 表示"......是天生的科学家"

### 例5.2（案例：学生与课程）

设：
- $$Student(x)$$：$$x$$ 是学生
- $$Course(y)$$：$$y$$ 是课程
- $$Takes(x, y)$$：$$x$$ 选修课程 $$y$$

公式：$$\forall x (Student(x) \to \exists y (Course(y) \land Takes(x, y)))$$

意思是："每个学生至少选了一门课程"。

## 5.4 语义

在命题逻辑中，可以使用真值表或真假赋值来计算一个公式的真值。其中，只需要关注命题符号的含义以及命题联结词的含义。在一阶逻辑中，还需要考虑关系符号、函数符号、变元符号、个体符号的含义。

这些符号的含义与**论域**有关。论域是包含所有被讨论对象的集合。在本书中，所有的论域都是**非空的**。

给定任意非空论域 $$D$$，**解释**是一个映射：
- 把个体符号映射为 $$D$$ 中的对象
- 把 $$n$$ 元函数符号映射为从 $$D^n$$ 到 $$D$$ 的函数
- 把 $$n$$ 元关系符号映射为 $$D$$ 上的 $$n$$ 元关系

另一方面，对于每个自由变元，可以把论域中的对象指派给它。因此，**指派**也是一个映射。在本书中，把解释和指派统称作**赋值**，记作 $$v$$。

### 定义5.4（赋值）

给定论域 $$D$$，则有：

- 对于个体常元 $$a$$，把它解释为论域中的个体，记作 $$v(a) \in D$$。
- 对于 $$n$$ 元函数符号 $$f$$，把它解释为从 $$D^n$$ 到 $$D$$ 的函数，记作 $$v(f): D^n \to D$$。
- 对于 $$n$$ 元谓词符号 $$F$$，把它解释为 $$D$$ 上的 $$n$$ 元关系，记作 $$v(F) \subseteq D^n$$。
- 对于自由变元 $$x$$，给它指派 $$D$$ 中的个体，记作 $$v(x) \in D$$。

通常把 $$v(a), v(f), v(F), v(x)$$ 分别记作 $$a^v, f^v, F^v, x^v$$。

### 核心概念理解：什么是"解释"与"指派"？

- **解释** = 建立"符号 $$\to$$ 世界"的映射。例如：个体常元 $$a$$ 表示"张三"，谓词 $$BornIn(x, y)$$ 表示"$$x$$ 出生在 $$y$$"。
- **指派** = 给变元具体的取值。例如：若给 $$x$$ 指派的值是"张三"，给 $$y$$ 指派的值是"北京"，那么 $$BornIn(x, y)$$ 的真假取决于"张三是否出生在北京"。

### 例5.3（案例：智能家居中的语义）

考虑句子"有设备亮着，且有设备已上锁"。设：
- 把 $$LightOn(x)$$ 解释为"$$x$$ 亮着"
- 把 $$Locked(y)$$ 解释为"$$y$$ 已上锁"

论域为所有设备。把"客厅灯"指派给 $$x$$，把"前门"指派给 $$y$$。若现实状态是"客厅灯亮着、前门已上锁"，则"有设备亮着，且有设备已上锁"（即 $$\exists x \exists y (LightOn(x) \land Locked(y))$$）在这一解释下为真。

### 例5.4（个体符号、函数符号、关系符号的解释）

给定公式 $$\forall x (F(x) \to \exists y (G(y) \land H(x, y)))$$ 和 $$F(g(ZS))$$，设 $$D = \{s_1, s_2, s_3, b_1, b_2, b_3\}$$。我们可以 $$D$$ 为论域，构作一个赋值 $$v$$，使得：

- $$ZS^v = s_1$$
- $$g^v = \{(s_1, s_2), (s_2, s_3)\}$$
- $$F^v = \{s_1, s_2, s_3\}$$
- $$G^v = \{b_1, b_2, b_3\}$$
- $$H^v = \{(s_1, b_1), (s_1, b_2), (s_2, b_1), (s_3, b_3)\}$$

依据定义5.4和例5.4可知，在一阶公式中，除了命题联结词，需要解释的元素包括个体符号、函数符号和关系符号。对于自由变元符号，可以把论域中的个体指派给它，这与对个体符号的解释不同。换句话说，在同一个解释下，个体符号所对应的论域中的个体是确定的，但自由变元符号可以有不同的指派。

### 定义5.5（项的值）

一阶逻辑语言的项在以 $$D$$ 为论域的赋值 $$v$$ 之下的值递归地定义如下：

1. $$a^v, x^v \in D$$
2. $$f(t_1, \ldots, t_n)^v = f^v(t_1^v, \ldots, t_n^v)$$

### 例5.5（项的值）

继续例5.4，我们有 $$g(ZS)^v = g^v(ZS^v) = g^v(s_1) = s_2$$。

### 符号约定

为了定义公式在赋值之下的真值，我们约定如下使用符号的规定。设 $$v$$ 是以 $$D$$ 为论域的赋值，$$a \in D$$，$$u$$ 是自由变元符号。我们用 $$v(u/a)$$ 表示一个以 $$D$$ 为论域的赋值，它除了 $$u^{v(u/a)} = a$$ 之外，对关系符号、函数符号、个体符号的解释以及对其他自由变元符号的指派和 $$v$$ 完全相同。

### 定义5.6（公式的真值）

一阶公式的真值可递归地定义如下：

- **原子公式**：
  $$F(t_1, \ldots, t_n)^v = \begin{cases} 1, & \text{如果 } (t_1^v, \ldots, t_n^v) \in F^v \\ 0, & \text{否则} \end{cases}$$

- **等值**：
  $$(t_1 \equiv t_2)^v = \begin{cases} 1, & \text{如果 } t_1^v \text{ 和 } t_2^v \text{ 是 } D \text{ 中相同的元素} \\ 0, & \text{否则} \end{cases}$$

- **否定**：
  $$(\neg\phi)^v = \begin{cases} 1, & \text{如果 } \phi^v = 0 \\ 0, & \text{否则} \end{cases}$$

- **合取**：
  $$(\phi \land \psi)^v = \begin{cases} 1, & \text{如果 } \phi^v = \psi^v = 1 \\ 0, & \text{否则} \end{cases}$$

- **析取**：
  $$(\phi \lor \psi)^v = \begin{cases} 1, & \text{如果 } \phi^v = 1 \text{ 或 } \psi^v = 1 \\ 0, & \text{否则} \end{cases}$$

- **蕴涵**：
  $$(\phi \to \psi)^v = \begin{cases} 1, & \text{如果 } \phi^v = 0 \text{ 或 } \psi^v = 1 \\ 0, & \text{否则} \end{cases}$$

- **等值**：
  $$(\phi \leftrightarrow \psi)^v = \begin{cases} 1, & \text{如果 } \phi^v = \psi^v \\ 0, & \text{否则} \end{cases}$$

- **全称量词**：
  $$\forall x\phi(x)^v = \begin{cases} 1, & \text{如果，由 } \phi(x) \text{ 构作 } \phi(u) \text{ (取 } u \text{ 不在 } \phi(x) \text{ 中出现), 对于任意的 } a \in D, \text{有 } \phi(u)^{v(u/a)} = 1 \\ 0, & \text{否则} \end{cases}$$

- **存在量词**：
  $$\exists x\phi(x)^v = \begin{cases} 1, & \text{如果，由 } \phi(x) \text{ 构作 } \phi(u) \text{ (取 } u \text{ 不在 } \phi(x) \text{ 中出现), 存在 } a \in D, \text{有 } \phi(u)^{v(u/a)} = 1 \\ 0, & \text{否则} \end{cases}$$

当 $$(t_1^v, \ldots, t_n^v) \in F^v$$，也记作 $$F^v(t_1^v, \ldots, t_n^v) = 1$$。

### 例5.6（公式的真值）

继续例5.4和5.5，我们有：

- $$F(g(ZS))^v = F^v(g(ZS)^v) = F^v(s_2) = 1$$，其中 $$s_2 \in F^v$$
- $$\forall x (F(x) \to \exists y (G(y) \land H(x, y)))^v = 1$$

为了验证 $$\forall x (F(x) \to \exists y (G(y) \land H(x, y)))^v = 1$$，由 $$F(x) \to \exists y (G(y) \land H(x, y))$$ 构作 $$F(u) \to \exists y (G(y) \land H(u, y))$$；由 $$G(y) \land H(u, y)$$ 构作 $$G(w) \land H(u, w)$$。

具体的推理过程如下（其中，$$\Leftrightarrow$$ 表示"当且仅当"）：

$$\forall x (F(x) \to \exists y (G(y) \land H(x, y)))^v = 1$$
$$\Leftrightarrow$$ 对于任意的 $$a \in D$$，$$(F(u) \to \exists y (G(y) \land H(u, y)))^{v(u/a)} = 1$$
$$\Leftrightarrow$$ 对于任意的 $$a \in D$$：$$F^v(a) = 0$$，或 $$(\exists y (G(y) \land H(u, y)))^{v(u/a)} = 1$$
$$\Leftrightarrow$$ 对于任意的 $$a \in D$$：$$F^v(a) = 0$$，或存在 $$b \in D$$，使得 $$(G(w) \land H(u, w))^{v(u/a, w/b)} = 1$$
$$\Leftrightarrow$$ 对于任意的 $$a \in D$$：$$F^v(a) = 0$$，或存在 $$b \in D$$，使得 $$G(w)^{v(u/a, w/b)} = 1$$ 且 $$H(u, w)^{v(u/a, w/b)} = 1$$

由于 $$D = \{s_1, s_2, s_3, b_1, b_2, b_3\}$$，如果 $$a = b_1$$ 或 $$a = b_2$$ 或 $$a = b_3$$，则 $$F^v(a) = 0$$ 成立；否则，当 $$a = s_1$$ 或 $$a = s_2$$ 时，取 $$b = b_1$$。当 $$a = s_3$$ 时，取 $$b = b_3$$。此时 $$G(w)^{v(u/a, w/b)} = 1$$ 且 $$H(u, w)^{v(u/a, w/b)} = 1$$ 成立。

## 5.5 逻辑推论

与命题逻辑对应，给定一组一阶公式集合 $$\Phi$$ 作为前提，我们希望知道 $$\Phi$$ 是否在逻辑上蕴涵某个结论 $$\phi$$。当对于任意的公式 $$\phi \in \Phi$$, $$\phi^v = 1$$，记作 $$\Phi^v = 1$$。

### 定义5.7（逻辑推论）

设 $$\Phi$$ 是一组公式集合，$$\phi$$ 是一个公式。==逻辑推论 $$\Phi \models \phi$$ 成立，当且仅当对于任何赋值 $$v$$，如果 $$\Phi^v = 1$$ 则 $$\phi^v = 1$$。==

### 例5.7（逻辑推论的证明）

设 $$\Phi = \{\forall x (F(x) \to \exists y (G(y) \land H(x, y))), F(g(ZS))\}$$ 为一组公式集合，$$\phi = \exists y (G(y) \land H(g(ZS), y))$$ 为一个公式。证 $$\Phi \models \phi$$。

**解**：设 $$\Phi \not\models \phi$$。于是，存在一个以某非空集合 $$D$$ 为论域的赋值 $$v$$，使得：

$$\forall x (F(x) \to \exists y (G(y) \land H(x, y)))^v = 1 \qquad \text{(5.1)}$$

$$F(g(ZS))^v = 1 \qquad \text{(5.2)}$$

$$\exists y (G(y) \land H(g(ZS), y))^v = 0 \qquad \text{(5.3)}$$

由 $$F(x) \to \exists y (G(y) \land H(x, y))$$ 构作 $$F(u) \to \exists y (G(y) \land H(u, y))$$；由 $$G(y) \land H(u, y)$$ 构作 $$G(w) \land H(u, w)$$；由 $$G(y) \land H(g(ZS), y)$$ 构作 $$G(w) \land H(g(ZS), w)$$。

由式（5.2）可得：$$F^v(g(ZS)^v) = 1$$。于是，存在 $$b = g(ZS)^v \in D$$，使得 $$F^v(b) = 1$$。

依据式（5.1）和 $$F^v(b) = 1$$，$$\exists y (G(y) \land H(u, y))^{v(u/b)} = 1$$。于是，存在 $$a \in D$$，使得 $$(G(w) \land H(u, w))^{v(u/b, w/a)} = 1$$，即 $$G^v(a) \land H^v(b, a) = 1$$。于是，$$G^v(a) = 1$$ 并且 $$H^v(b, a) = 1$$。

由式（5.3）可得：对于任意的 $$a \in D$$，$$\neg G^v(a) = 1$$ 或 $$\neg H^v(g(ZS)^v, a) = 1$$。于是，对于任意的 $$a \in D$$，$$G^v(a) = 0$$ 或 $$H^v(b, a) = 0$$。这与 $$G^v(a) = 1$$ 并且 $$H^v(b, a) = 1$$ 矛盾。因此，假设不成立，$$\Phi \models \phi$$ 得证。

### 可满足性和有效性

设 $$\psi$$ 是一个一阶公式：

- $$\psi$$ 是**有效的**，即 $$\models \psi$$，当且仅==当对于任何论域 $$D$$ 下任何赋值 $$v$$，$$\psi^v = 1$$。==
- $$\psi$$ 是**可满足的**，当且仅当==存在某个论域 $$D$$ 下的赋值 $$v$$ 使得 $$\psi^v = 1$$。==

### 一阶逻辑中的常见语义等价关系

- $$\neg \forall x \phi(x) \models\mid \exists x \neg \phi(x)$$
- $$\neg \exists x \phi(x) \models\mid \forall x \neg \phi(x)$$
- $$\forall x [\phi(x) \land \psi(x)] \models\mid \forall x \phi(x) \land \forall x \psi(x)$$
- $$\exists x [\phi(x) \lor \psi(x)] \models\mid \exists x \phi(x) \lor \exists x \psi(x)$$
- $$\phi \land \forall x \psi(x) \models\mid \forall x (\phi \land \psi(x))$$，$$x$$ 不在 $$\phi$$ 中出现
- $$\phi \land \exists x \psi(x) \models\mid \exists x (\phi \land \psi(x))$$，$$x$$ 不在 $$\phi$$ 中出现
- $$\phi \lor \forall x \psi(x) \models\mid \forall x (\phi \lor \psi(x))$$，$$x$$ 不在 $$\phi$$ 中出现
- $$\phi \lor \exists x \psi(x) \models\mid \exists x (\phi \lor \psi(x))$$，$$x$$ 不在 $$\phi$$ 中出现

### 例5.8（案例：法律推理）

- 前提1：$$\forall x (Theft(x) \to Crime(x))$$（所有盗窃行为构成犯罪）
- 前提2：$$Theft(zhangsan)$$（张三实施盗窃）
- 结论：$$Crime(zhangsan)$$（张三构成犯罪）

逻辑上：$$\{\forall x (Theft(x) \to Crime(x)), Theft(zhangsan)\} \models Crime(zhangsan)$$

### 例5.9（案例：医学推理）

- 前提2：$$Covid(lisi)$$（李四感染新冠）

## 5.6 前束范式

在把一阶逻辑中的公式转化为范式时，先把它们变换为语义上等值的前束范式。

### 定义5.9（前束范式）

称一阶逻辑公式 $$\phi$$ 为**前束范式**，当且仅当它有如下形式：

$$Q_1 x_1 \ldots Q_n x_n \phi'$$

其中的 $$Q_1 \ldots Q_n$$ 是 $$\forall$$ 或 $$\exists$$，并且 $$\phi'$$ 不含量词。我们称 $$Q_1 x_1 \ldots Q_n x_n$$ 为**前束词**，称 $$\phi'$$ 为**母式**。

前束范式的母式可以进一步变换为合取范式或析取范式。

### 核心概念理解：什么是前束范式？

一个公式可能在不同地方夹杂量词。前束范式就是把所有量词"前移"到公式开头，后面只留下没有量词的核心部分。

例如，公式 $$\forall x (F(x) \to \exists y G(y, x))$$，其前束范式为 $$\forall x \exists y (F(x) \to G(y, x))$$。

一般地，**任何一阶逻辑公式都可以等价地化为前束范式**。

### 例5.10（把公式变换为前束合取范式）

把公式 $$[\neg \exists x_1 F(x_1) \lor \forall x_2 G(x_2)] \land [F(x_3) \to \forall x_4 H(x_4)]$$ 变换为前束合取范式。

**解**：依据公式的等价关系，可得到如下变换过程：

$$\begin{align}
&[\neg \exists x_1 F(x_1) \lor \forall x_2 G(x_2)] \land [F(x_3) \to \forall x_4 H(x_4)] \\
\models\mid\ & [\forall x_1 \neg F(x_1) \lor \forall x_2 G(x_2)] \land [\neg F(x_3) \lor \forall x_4 H(x_4)] \\
\models\mid\ & \forall x_1 \forall x_2 \forall x_4 ([\neg F(x_1) \lor G(x_2)] \land [\neg F(x_3) \lor H(x_4)])
\end{align}$$

## 5.7 消解原理

在一阶逻辑中，需要处理公式中的变元和量词。与命题逻辑类似，也需要把公式转化为等价的子句形式。

当对包含自由变元的子句进行消解时，如果这些子句都是全称量化的，则可以去掉量词。因此，为简单起见，我们先考虑**不含存在量词的前束范式**。

这时，子句形式与之前命题逻辑的一样，但原子公式是 $$F(t_1, \ldots, t_n)$$。同时，对子句的理解与之前一样，但变元被理解成全称量化。

### 例5.11（用子句表示公式）

$$\{[P(x), \neg R(a, f(b, x))], [Q(x, y)]\}$$ 表示的是：

$$\forall x \forall y ([P(x) \lor \neg R(a, f(b, x))] \land Q(x, y))$$

### 一阶消解中的合一

在一阶消解中，需要考虑如何把消解规则应用于包含变元的子句。由于包含变元的子句是全称量化的，在证明可满足性时，可以使用这些子句的实例。

例如，假设我们有如下子句：

$$[F(x, a), \neg G(x)]$$ 和 $$[\neg F(b, y), \neg H(b, f(y))]$$

那么，至少潜在地，我们有如下子句：

$$[F(b, a), \neg G(b)]$$ 和 $$[\neg F(b, a), \neg H(b, f(a))]$$

这两个子句消解于 $$[\neg G(b), \neg H(b, f(a))]$$。

### 核心概念理解：一阶消解 vs. 命题消解

- 在命题逻辑中，消解就是"$$p$$ 和 $$\neg p$$ 抵消"；
- 在一阶逻辑中，还涉及变元和项，需要做**合一**处理。

例如：对于 $$P(x)$$ 与 $$\neg P(a)$$，可以通过把 $$P(x)$$ 中的 $$x$$ 代换为 $$a$$，从而实现合一并完成消解。

### 定义5.10（一阶消解推演规则）

给定两个子句 $$c_1 \cup \{L_1\}$$ 和 $$c_2 \cup \{\bar{L_2}\}$$，如果它们没有公共变元，且存在一个代换 $$\theta$$，使得 $$L_1\theta = \bar{L_2}\theta$$，那么可以推出子句 $$(c_1 \cup c_2)\theta$$。这时我们说 $$\theta$$ 是 $$L_1$$ 和 $$\bar{L_2}$$ 的**合一**。

通过运用这条规则，消解推演的定义与命题逻辑的相同。同时，$$\Phi \vdash_{Res} \bot$$ 当且仅当 $$\Phi \models_{Res} \bot$$（证明略）。

### 例5.12（消解推演过程）

设有一个知识库 KB 包含如下知识：
- $$\forall x (GradStudent(x) \to Student(x))$$
- $$\forall x (Student(x) \to Hardworker(x))$$
- $$GradStudent(li\_ming)$$

**解**：关于 $$KB \cup \{\neg Hardworker(li\_ming)\} \vdash_{Res} \bot$$ 的消解推演过程如下：

| 步骤 | 子句 | 依据 |
|:---:|------|------|
| 1 | $$[\neg GradStudent(x), Student(x)]$$ | (前提) |
| 2 | $$[\neg Student(x), Hardworker(x)]$$ | (前提) |
| 3 | $$[GradStudent(li\_ming)]$$ | (前提) |
| 4 | $$[\neg Hardworker(li\_ming)]$$ | (前提) |
| 5 | $$[\neg Student(li\_ming)]$$ | (消解, 2, 4, $$\{x/li\_ming\}$$) |
| 6 | $$[\neg GradStudent(li\_ming)]$$ | (消解, 1, 5, $$\{x/li\_ming\}$$) |
| 7 | $$[\ ]$$ | (消解, 3, 6) |

## 5.8 Prolog

Prolog 程序的基本构成与霍恩子句逻辑程序完全相同，它们由三个部分组成：**事实**、**规则**和**问题**。

### 一、事实

事实是关于客体性质和关系的事实语句。事实用原子公式表示，与霍恩子句不同的是，Prolog 不用大写字母给谓词起头，常量全部都用小写字母或字母串表示，大写字母专用于表示变量。另外，Prolog 不在事实后使用符号 $$\leftarrow$$，而使用句号。事实语句中的变元表示该事实对于任何客体均成立。在 Prolog 中，数据库是事实的集合。Prolog 从这些事实中推出知识。

### 二、规则

规则表达的是客体的性质或关系。在 Prolog 中，规则"如果 $$q_1, \ldots, q_m$$ 那么 $$p$$"被表示为：

$$p :- q_1, \ldots, q_m$$

其中，$$:-$$ 的左部称为**规则头**，$$:-$$ 的右部称为**规则体**。规则头为单一的原子公式，规则体则可以是 $$q_1, \ldots, q_m$$ 形，$$q_i$$ 为原子公式，它们之间是合取关系。

### 三、问题

问题指关于客体性质或关系的问题。问题是程序执行的目标。当我们把事实和规则看作公理，那么问题就是待证的命题；当我们把事实和规则看作知识库，那么问题就是利用这些知识求解的目标。

### Prolog 程序的执行方式

Prolog 程序的执行方式与霍恩子句逻辑程序的执行方式是一致的，主要操作是**搜索**、**匹配**与**回溯**。与目标中的项匹配的事实和规则是在程序中自上而下地搜索得到的。当目标中的一项失败时，如果目标中有已成功的项，那么就重新调用这些成功项中最右边的一个，谋求新的成功，这就是**回溯**。这里的项指目标中的一个过程调用，与一阶语言中的项的概念不同。

### 核心概念理解：Prolog 的思路

传统程序员告诉计算机"怎么做"；Prolog 程序员只需要告诉计算机"事实和规则"，然后问问题，系统会自动推出答案。因此 Prolog 是一种**声明式编程语言**。

### 例5.13（Prolog 程序）

下列事实、规则和问题组成一个 Prolog 程序：

```prolog
likes(li_ming, sports).
likes(wang_fang, zhang_wei).
likes(wang_fang, sports).          % 事实
likes(chen_qiang, zhang_wei).

friend(chen_qiang, X) :- likes(X, sports), likes(X, zhang_wei).  % 规则

?- friend(chen_qiang, Y).          % 问题
```

它的执行过程大体是：

1. 自上而下搜索，找到规则（5）与问题（6）匹配，得到目标：
   ```
   ?- likes(X, sports), likes(X, zhang_wei)
   ```
   变元 $$Y$$ 与 $$X$$ 共享（因为代换为 $$\{X/Y\}$$）。

2. 目标中第一个项 $$likes(X, sports)$$ 与自上而下搜索到的事实(1)匹配，成功，$$X$$ 被命名（赋值）为 $$li\_ming$$（$$Y$$ 同时取得值 $$li\_ming$$），从而目标中第二个项成为 $$likes(li\_ming, zhang\_wei)$$，它是现在的目标中的唯一项，然而它匹配失败。

3. 于是产生**回溯**，重新执行目标 $$likes(X, sports)$$，这次它不与(1)匹配，而由(2)开始向下搜索，得到事实(3)与之匹配，成功，$$X$$ 被命名为 $$wang\_fang$$，$$Y$$ 共享这个命名。现在目标项 $$likes(X, zhang\_wei)$$ 成为项 $$likes(wang\_fang, zhang\_wei)$$。

4. 自上而下的搜索得到事实(2)与之匹配，成功。至此，程序执行完毕，目标成功，输出 $$Y = wang\_fang$$。

### 例5.14（应用案例：AI中的 Prolog）

- **专家系统**：早期医学诊断系统 MYCIN 使用类似逻辑规则进行推理。
- **自然语言处理**：句法分析器可用 Prolog 实现，利用规则描述语法结构。
- **知识推理**：通过事实与规则，自动回答查询或验证一致性。

### 语言层面
一阶逻辑语言比命题逻辑更强大，能够刻画对象、属性、关系与量化。它是人工智能中知识表示与推理的核心语言，支撑专家系统、知识图谱、数据库查询、自然语言理解等任务。

### 语义层面
语义让符号"落地"，使一阶逻辑公式能够与现实世界对应。在AI中，语义解释是知识图谱建模（实体 + 关系）、数据库查询（SQL $$\to$$ 逻辑公式）、自动问答系统等任务的基础。

### 推理机制
一阶逻辑的语义推论扩展了命题逻辑，能够处理涉及对象、属性和关系的推理。有效性与可满足性之间的判定关系（$$\Phi \models \phi \iff \Phi \cup \{\neg\phi\}$$ 不可满足）是自动推理的基本方法。应用于法律推理、医学诊断、知识库一致性检查、专家系统等。

### 范式与计算
前束范式将量词前置，为算法处理提供标准化形式，是合取范式转换与消解推理的前提步骤。广泛应用于自动定理证明、逻辑编程、知识库检验等任务。

### 消解原理
消解规则通过寻找互补文字并结合合一算法处理变量，把复杂推理统一为可计算的形式操作。它是逻辑编程（Prolog）、知识推理、自动定理证明的基石。

### 逻辑编程
Prolog 将"程序 = 知识库 + 查询"，以消解与合一结合回溯搜索实现自动推理。在专家系统、自然语言处理、知识图谱查询、智能问答等场景中具有广泛应用。

### 哲学与方法论启示
一阶逻辑揭示了知识建模中"对象--属性--关系"的形式结构，语义解释展示了符号如何对应现实，推理机制体现了理性如何在复杂结构中保持一致性。它不仅是人工智能的核心工具，也为哲学中关于知识、真理与理性的讨论提供了形式化方法论。
