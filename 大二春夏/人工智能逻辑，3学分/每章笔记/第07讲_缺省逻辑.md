# 第7讲 缺省逻辑

---

## 概述

为了处理不完备信息，可以对经典一阶逻辑进行扩展，建立非单调逻辑。在现有文献中，有多种非单调逻辑，包括缺省逻辑 (Default Logic)、自认知逻辑 (Autoepistemic Logic)、限定逻辑 (Circumscription)、回答集编程 (Answer Set Programming) 等。在本书中，我们以缺省逻辑为例，阐明这种典型的非单调逻辑的推理机制。

缺省逻辑由 **莱特 (Ray Reiter)** 于1980年提出，是一种可以处理例外情况的非单调形式体系。例外情况的存在源于信息的不完备性。例如，对于常识性知识"通常情况下，熊科动物以肉为主食"，存在例外情况"大熊猫是熊科动物，但不以肉为主食"。

**缺省推理最基本的特性是暂时性**：即在缺少信息的情况下推出暂时的结论。由于信息的缺失，由缺省推理得到的结论可能是错误的。当出现新信息时，已经推出的错误结论可能被推翻。我们把推理系统的这个特性称为**非单调性** (Non-monotonicity)。该特性是缺省推理研究的核心。

缺省推理的另一个著名特征是它在本质上是**定性的** (Qualitative)。缺省推理只处理命题的真值判断，而不处理不确定性程度。因此，缺省推理支持信念的非单调修正。

缺省逻辑通过扩充经典逻辑来实现对不完备信息的表示与推理，其方式是引入特殊的推理规则——**缺省规则** (Default Rule)。

---

## 7.1 缺省规则

语句"通常情况下，如果 φ 为真，那么 ψ 为真"所表达的是在没有出现例外的情况下"如果 φ 为真，那么 ψ 为真"。换句话说，此处的"通常情况下"使得这一缺省规则并非在所有情况下都是可应用的，例外情况的出现可以阻止它的可应用性。

为了明确表示例外，把缺省规则表示为：

```
φ : ψ / η
```

读作"如果可以证明 φ，且 ψ 是一致的（或者，等价地，不能证明 ¬ψ，即：例外 ¬ψ 没有发生），那么可以推出 η。"

### 核心概念理解：缺省规则的三部分

- **先决条件 (Prerequisite) φ**：已知的事实
- **缺省条件 (Justification) ψ**：只要它与现有知识不矛盾，就可以暂时假设
- **结论 (Consequent) η**：在前提与正当性条件成立时得到的推论

换句话说：**缺省规则 = "在正常情况下，我们可以这样认为"**。

我们通过如下几个例子展示缺省规则的用法。

### 例7.1（案例：比赛与恶劣天气）

北京冬奥会组委会采用这样一条规则："如果安排了比赛项目（p），那么将如期举行（q），除非遭遇恶劣天气（r）"。

**经典逻辑的局限性分析**：

上述规则在经典逻辑中被形式化为：

```
p ∧ ¬r → q
```

这种形式化的主要问题在于，其应用必须以"不会遭遇恶劣天气（¬r）"的确定性信息为前提。因此，在天气状况不确定的情况下，系统将无法安排任何比赛。这种对确定性的要求，阻碍了其借助"可假定无恶劣天气"这一缺省条件作出合理推断。

**使用缺省逻辑的表示**：

```
p : ¬r
------       (7.1)
  q
```

该规则的含义是：如果已安排比赛项目，且没有信息表明比赛期间将遭遇恶劣天气，则可默认天气条件适宜，并推出比赛将如期举行的结论。然而，如果在赛前获得天气预报显示极端恶劣天气，则上述假定不再成立。此时，该缺省规则无法被应用，因此不能推出比赛将如期举行的结论。

**另一种等价表示**：若是系统中包括经典蕴涵式 `r → ¬q`，则上述缺省规则可以表示为：

```
p : q
-----        (7.2)
  q
```

如果我们知道遭遇恶劣天气了，那么可以推出比赛不能举行，即 ¬q，因此依据该缺省规则的要求，不能假定 q。在这个表达中，缺省规则的含义是：比赛通常会如期举行，而这条规则的例外被表示为上面的经典蕴涵式。

### 缺省规则的多种用途

缺省规则可以用于各种包含例外的知识表示。

#### (1) 原型推理 (Prototype Reasoning)

意指一个概念的大多数实例都具有某个特性。具体地，"典型地，孩子有父母"可以表示为：

```
C(x) : P(x)
-----------        (7.3)
  P(x)
```

其中，`C(x)` 表示 x 是孩子，`P(x)` 表示 x 有父母。

#### (2) 无风险推理 (No-Risk Reasoning)

其可以描述这样一种情境：即使一个结论不是最可能的，也可以推出它，因为另一种结论是灾难性的。例如，在法律领域，一般采用**无罪推定原则**"在没有反面证据时，被告人无罪"。这个原则可以表示为如下缺省规则：

```
A(x) : ¬I(x)
------------        (7.4)
  ¬I(x)
```

其中，`A(x)` 表示 x 是被告人，`I(x)` 表示 x 有罪。

#### (3) 法律推理中的例外条款

在法律推理领域，许多法律条款的适用性存在例外，因此通常可以把法律条款表示为缺省规则。依据我国法律，如果某人（防卫人）面临不法侵害，并且采取行为制止该不法侵害，同时没有明确证据表明该行为是以防卫为名行不法侵害之实，那么在没有其他相反信息的情况下，应默认该行为属于正当防卫。

这一信息可由如下缺省规则加上一个经典蕴涵式 `∀x(G(x) → ¬D(x))` 来表示：

```
U(x) ∧ S(x) : D(x)
-------------------        (7.5)
       D(x)
```

其中：
- `U(x)` 表示 x 面临不法侵害
- `S(x)` 表示 x 采取行动制止该不法侵害
- `D(x)` 表示 x 的行为属于正当防卫
- `G(x)` 表示 x 是以防卫为名行不法侵害之实

#### (4) 封闭世界假设 (Closed World Assumption, CWA)

它指的是这样的假定：如果一个句子为真，那么知道它为真。因此，当前不知道为真的句子都被假定是假的。依据这个假设，可以把一个应用领域描述为一些公理的集合，并以如下方式来推理：若一个命题不能从公理推出，则它是假的。运用缺省规则，封闭世界假设可以表示为：

```
⊤ : ¬φ
------        (7.6)
  ¬φ
```

对于每个原子公式 φ，这个缺省规则可以被解释为：如果可以一致地假设 ¬φ，那么可以得到 ¬φ。

### 例7.2（应用案例：医学诊断中的缺省规则）

在医学专家系统中，可以设置缺省规则：

```
Fever(x) : Flu(x)
-----------------
     Flu(x)
```

含义：如果 x 发烧，并且没有证据表明是其他疾病，可以缺省地认为他得了流感。若后续检测出"新冠阳性"，则该结论立即被推翻。

## 7.2 缺省逻辑的语法

本节介绍缺省逻辑的语言、缺省规则以及缺省理论的定义。

### 定义7.1（缺省逻辑的语言）

任何一阶语言是缺省逻辑的语言。因此，在缺省逻辑中，所用到的项、公式等的定义与一阶语言里的相同。

### 定义7.2（缺省规则）

一条缺省规则可以被形式化地表示为：

```
δ = φ : ψ1, ..., ψn / χ
```

其中，`φ, ψ1, ..., ψn` 是一阶逻辑公式，`n ≥ 0`。

- 公式 `φ` 称为**先决条件** (Prerequisite)
- `ψ1, ..., ψn` 是**缺省条件** (Justifications)
- `χ` 是 δ 的**结论** (Consequent)

有时，用 `pre(δ)` 表示 φ，用 `just(δ)` 表示 `{ψ1, ..., ψn}`，用 `cons(δ)` 表示 χ。

对于缺省规则集合 `D`，用 `cons(D)` 表示 D 中缺省的结论集合。

为简便起见，缺省规则也可表示为 `φ : ψ1, ..., ψn / χ`。

这种推理规则可以说是传统逻辑中规则 `φ / χ` (公式7.7) 的"**非单调化**"，因为该规则可能存在许多例外，引入各个 ψi 就是为了指出这些例外，使它们随着知识的增长而改变对特定命题的判断，从而改变推理的结果。

在缺省条件中的 ψi 表示就现有知识而言 ψi 可能成立，即 ¬ψi 尚未出现。换句话说，在 φ 与 χ 之间存在一定的间隙，即 φ 的真不能完全确保 χ 的真。因此，从 φ 到 χ 的推理属于**"跳到结论" (Jump to Conclusion)**。

#### 开规则与闭规则

对于一条缺省规则，如果其中的公式不都是句子（即含有自由变元），那么称之为**开规则**。

==把每个不含变元符号的项称为**基项** (Ground Term)。把一条开规则中的所有自由变元都替换为基项，得到的是一条**闭规则**。==

因此，可以把开规则看作是**缺省规则模式** (Default Rule Schema)，意指它表示一组缺省规则的集合。这个集合可以是无穷的。

给定一条缺省规则模式 `δ = φ : ψ1, ..., ψn / χ`，对于任意基代换 θ，可以得到如下闭规则：

```
δθ = φθ : ψ1θ, ..., ψnθ / χθ        (7.8)
```

### 例7.3（关于"熊科动物以肉为主食"的缺省规则表示）

给定一条缺省规则模式：

```
δ1 = bear(x) : meat(x) / meat(x)
δ2 = panda(x) : ¬meat(x) / ¬meat(x)
```

以及事实 `bear(tuantuan)` 和 `panda(tuantuan)`。

设 `θ = {x/tuantuan}`，可以得到如下两条闭规则：

```
δ1θ = bear(tuantuan) : meat(tuantuan) / meat(tuantuan)
δ2θ = panda(tuantuan) : ¬meat(tuantuan) / ¬meat(tuantuan)
```

### 定义7.3（规范缺省规则）

当一条缺省规则的缺省条件与结论相同时，称之为**规范**的 (Normal)，形如 `φ : χ / χ`。

在上一节的例子中，规范缺省包括规则式(7.2)-(7.4)和(7.6)。

### 定义7.4（缺省理论）

一个**缺省理论** (Default Theory) 是一个二元组 `T = ⟨W, D⟩`，其中：

- `W` 是一阶逻辑公式集合，用于表示已知的或约定的事实集合
- `D` 是一个可数的缺省规则集合

当 D 中所有规则是闭规则时，称理论 `T = ⟨W, D⟩` 是**闭理论** (Closed Theory)。

### 缺省理论的非单调性

考虑理论 `T1 = ⟨W1, D1⟩`，其中：

```
D1 = { ⊤ : p / q }
W1 = ∅
```

那么，q 在 T1 中可推出。

如果把 ¬p 添入 W1，那么将使 W1 成为 `W1' = {¬p}`，并使 T1 成为 `T1' = ⟨W1', D1⟩`。这时，尽管 T1' 是 T1 的扩充，但 q 却不能从 T1' 中推出。

**核心概念理解：缺省理论的组成**：

一个缺省理论就是"事实 + 缺省规则"的组合：

- **W** = 我们已经知道的确定事实
- **D** = 在没有矛盾证据时可以使用的缺省规则

## 7.3 缺省逻辑的语义

给定一个缺省理论，其语义指的是可以接受或相信哪些语句。被接受或相信的语句集合称为**外延** (Extension)。

**核心概念理解：什么是外延？**

外延 = "一个可能的、自洽的世界"：

- 它必须包含所有背景知识
- 它还包含那些在缺省规则支持下、且不与已有知识矛盾的结论

不同的外延对应不同的假设选择，因此缺省逻辑可能有**多个外延**。

### 例7.4（应用案例：AI中的多假设推理）

在专家系统或常识推理中，系统可能面对多种合理的推论路径：

- 医学诊断：发烧 -> 流感？还是新冠？
- 法律推理：是否适用某个例外条款？

缺省逻辑的外延机制允许系统同时考虑多个可能结论，并在新证据出现时选择或修正。

---

### 7.3.1 外延的不动点定义

对于一个缺省理论，可接受的信念满足以下直觉：

1. **事实是可以被相信的**，这一点在直觉上显然成立
2. **如果一条缺省规则在某个背景下可以被应用，那么它的结论也是可以被相信的**
3. **如果相信了一组语句（包含事实以及由应用缺省规则而添加的结论），那么也相信这组语句的所有演绎结论**，这一点是为了确保一阶逻辑的演绎封闭性
4. **除了上述三点，不相信其他无关的语句**

下面对第二点做进一步讨论。

关于如何应用缺省规则来扩充结论，考虑规则 `φ : ψ1, ..., ψn / χ`。它的非形式含义是：如果相信 φ，并且可以一致地假设 `ψ1, ..., ψn`，那么可以相信 χ。

为了形式化这个解释，我们必须说在什么**背景**下可以相信 φ，以及 `ψ1, ..., ψn` 应该与什么一致。这时，我们也说这条缺省规则是在这个背景下**可应用的**。

### 例7.5（把事实集合作为缺省规则应用的背景）

依据例7.3，可以构造缺省理论 `T2 = ⟨W2, D2⟩`，其中：

```
W2 = {bear(tuantuan), panda(tuantuan)}
D2 = {δ1', δ2'}
```

其中：

```
δ1' = bear(tuantuan) : meat(tuantuan) / meat(tuantuan)
δ2' = panda(tuantuan) : ¬meat(tuantuan) / ¬meat(tuantuan)
```

首先，可以相信两个事实 `bear(tuantuan)` 和 `panda(tuantuan)`。其次，可以通过缺省规则来扩充信念。

一种最直观的方法是把**事实集合作为背景**。但采用这种方法明显存在问题：对于例7.3中的两条规则，当把事实集合作为背景时，都是可应用的：

- 对于第一条规则，由于相信 `bear(tuantuan)`，且假设 `meat(tuantuan)` 与事实集合没有发生不一致，因此可以相信 `meat(tuantuan)`
- 类似地，可以相信 `¬meat(tuantuan)`

这样，就同时相信了一对不一致的结论 `meat(tuantuan)` 和 `¬meat(tuantuan)`。

**问题分析**：尽管有两条缺省规则存在矛盾，但同时应用它们而得到矛盾却与直觉不符。换句话说，对于两条相互矛盾的规则，一旦应用了其中一条，另一条便不应再被应用。具体来说，如果先应用第一条规则，并基于当前所得知识来检查第二条规则的可应用性，那么第二条规则将因不满足条件而被阻止应用：当应用了第一条规则，我们知道"团团以肉为主食"，因此与第二条规则的假设"团团不以肉为主食"不一致。

**结论**：为了避免在应用规则后产生不一致的结论，把背景设置为事实集合是不合适的，还需要考虑在应用规则之后各条规则的结论对其他规则的缺省条件的影响。

### 定义7.5（缺省规则的应用）

给定两个句子集合 Φ 和 Ψ，以及一条缺省规则 `δ = φ : ψ1, ..., ψn / χ`，我们说**在 Φ 下把 δ 应用于 Ψ**，意指：

如果 `φ ∈ Ψ`，而且 `¬ψ1 ∉ Φ, ..., ¬ψn ∉ Φ`，那么 `χ ∈ Ψ`。

当 Φ = Ψ 时，我们也说把 δ 应用于 Ψ。

在这个定义中：

- **Φ** 是一组语句集合，用于检查规则可应用性时的缺省条件成立与否
- **Ψ** 表示可接受的语句集合

这样，给定一个缺省理论 `T = ⟨W, D⟩` 和一组语句集合 Φ，通过所有缺省规则，可以得到一组可接受的语句集合 Ψ。现在的问题是，给定 T 和 Φ，如何产生 Ψ，使之满足上述的四个条件？一种方法是去定义一个**算子 Γ**，它作用于 Φ，产生语句集合 Γ(Φ)，使其满足上述四个条件。

### 定义7.6（极小扩充语句集合）

设 `T = ⟨W, D⟩` 是一个缺省理论，Γ 为关于 D 的一个算子，Γ 作用于任意的语句集合 Φ，产生满足下列三个性质的**极小语句集合** `Γ(Φ)`：

1. **W ⊆ Γ(Φ)**：事实集合被包含
2. **Th(Γ(Φ)) = Γ(Φ)**：Γ(Φ) 是演绎封闭的。这里 `Th(Γ(Φ))` 为语句集 `{φ | Γ(Φ) ⊢ φ}`
3. 对于 D 中的任何缺省规则 `δ = φ : ψ1, ..., ψn / χ`，在 Φ 下把 δ 应用于 Γ(Φ)，即：如果 `φ ∈ Γ(Φ)`，而且 `¬ψ1 ∉ Φ, ..., ¬ψn ∉ Φ`，那么 `χ ∈ Γ(Φ)`

### 例7.6（极小扩充语句集合）

继续例7.5，其中 `W2 = {bear(tuantuan), panda(tuantuan)}`。

设有三个语句集合：

- `Φ1 = Th(W2)`
- `Φ2 = Th(W2 ∪ {meat(tuantuan)})`
- `Φ3 = Th(W2 ∪ {¬meat(tuantuan)})`

则：

- `Γ(Φ1) = Th(W2 ∪ {meat(tuantuan), ¬meat(tuantuan)})` — 包含矛盾命题，在直觉中是不可接受的
- `Γ(Φ2) = Th(W2 ∪ {meat(tuantuan)})` — 在直觉上是可接受的
- `Γ(Φ3) = Th(W2 ∪ {¬meat(tuantuan)})` — 在直觉上是可接受的

当存在矛盾命题时，任意一阶公式都属于演绎闭包。这时，从应用缺省规则得到的命题集合是"**不稳定的**"。

为了获得**稳定外延**，可以通过**不动点** (Fixed Point) 的概念来实现。

### 定义7.7（外延的不动点定义）

句子集合 E 是缺省理论 `T = ⟨W, D⟩` 的一个**外延**当且仅当 E 是算子 Γ 的一个不动点，即：

```
Γ(E) = E
```

### 例7.7（有两个不动点的缺省理论）

对于例7.6，Φ2 和 Φ3 是例7.5中缺省理论 T2 的两个外延。

即：

- 外延1：`Th(W2 ∪ {meat(tuantuan)})` — 团团以肉为主食
- 外延2：`Th(W2 ∪ {¬meat(tuantuan)})` — 团团不以肉为主食

### 例7.8（无不动点的缺省理论）

设 `T3 = ⟨W3, D3⟩`，其中：

```
D3 = { ⊤ : p / ¬p }
W3 = ∅
```

那么，T3 **无外延**，因为可以证明关于 D3 的算子 Γ 没有不动点。

证明（反证法）：若不然，设 E 为 Γ 不动点。考虑 `¬p ∈ E` 否？

- 如果 `¬p ∉ E`，那么根据定义7.6之(3)，`¬p ∈ E`（矛盾）
- 如果 `¬p ∈ E`，则依据定义 `¬p ∉ E`（矛盾）

两种情况下都产生矛盾。故原命题成立。

### 例7.9（有唯一不动点的缺省理论）

设 `T4 = ⟨W4, D4⟩`，其中：

```
D4 = { ⊤ : p / ¬q,  ⊤ : q / ¬r,  ⊤ : r / ¬s }
W4 = ∅
```

那么，T4 有唯一的外延 `Th({¬q, ¬s})`。

### 例7.10（有两个不动点的缺省理论）

设 `T5 = ⟨W5, D5⟩`，其中：

```
D5 = { p ∧ q1 : r1 / r1,  p ∧ q2 : r2 / r2 }
W5 = {p, q1 ∧ q2, r1 ↔ ¬r2}
```

那么，T5 有两个外延：

- `E1 = Th(W5 ∪ {r1})`
- `E2 = Th(W5 ∪ {r2})`

对于 E1，由于 `Th(W5 ∪ {r1})` 中有 `¬r2`，从而 `Γ(Th(W5 ∪ {r1}))` 中没有 r2，因为缺省规则 `p ∧ q2 : r2 / r2` 中的缺省条件 r2 不能成立。

### 总结：外延的存在性和唯一性

上述例子表明，**并非所有缺省理论都有外延，而且即便存在外延，也未必唯一**。

### 准归纳的外延定义

由于定义7.7所给出的外延是非建构性的，如何验证一个外延困难较大。如下**准归纳的外延定义**具有构造性的特点，它准确说明了信念在什么意义上基于 W 和 D，且对于证明非常有用。之所以称为"准归纳"，是因为外延 E 出现在归纳步骤之中。

设 E 为一阶命题集，`T = ⟨W, D⟩` 为一闭的缺省理论。递归定义 Ei (i = 0, 1, 2, ...) 如下：

```
E0 = W

Ei+1 = Th(Ei) ∪ { χ ∣ (φ : ψ1, ..., ψn / χ) ∈ D, φ ∈ Ei, ¬ψj ∉ E, 1 ≤ j ≤ n }
```

那么，**E 是 T 的一个外延当且仅当 `E = ∪[i=0 to ∞] Ei`**。

### 例7.11（由准归纳定义计算外延）

设 `T6 = ⟨W6, D6⟩`，其中：

```
D6 = {
    ⊤ : p / q,
    ⊤ : ¬p / r,
    q : s / s,
    s : q / p,
    r : ¬q / ¬p
}
W6 = ∅
```

那么，`E = Th({p, q, s})` 和 `E' = Th({¬p, r})` 是 T6 的两个外延。

**验证外延 E = Th({p, q, s})**：

- E0 = ∅
- E1 = {q} （应用第一条规则 ⊤:p/q 于 E0）
- E2 = Th({q}) ∪ {s} （应用第三条规则 q:s/s 于 E1）
- E3 = Th({q, s}) ∪ {p} = Th({p, q, s}) = E （应用第四条规则 s:q/p 于 E2）

同理，可以验证 E' 是 T6 的外延。

### 例7.12（不一致的外延）

设 `T7 = ⟨W7, D7⟩`，其中：

```
D7 = { p : r / r,  q : s / s }
W7 = {p ∨ q, ¬p, ¬q}
```

那么，T7 有唯一的外延 `Th(W7)`。该外延包含所有的一阶命题，且是**不一致的**。

一般地，一个闭缺省理论 `T = ⟨W, D⟩` 有**不一致的外延 E，当且仅当 W 不一致**。

### 7.3.2 外延的操作定义

外延的不动点定义不是完全建构性的，因此，为了获得一个缺省理论的外延，需要假定可能的命题集合，再从这些命题集合出发，判断哪些集合是外延。

现在来介绍一种建构性的外延定义方法，称为**外延的操作定义** (Operational Definition)。该方法通过缺省规则的**实施过程树**来定义外延。

#### 基本约定

给定一个缺省理论 `T = ⟨W, D⟩`，把一个缺省规则的序列记作 `Π = (δ0, δ1, ...)`，其中每条规则最多在 Π 中出现一次。这个序列可以看作是规则实施的顺序。

为了定义基于过程的外延，采用如下符号约定：

- 用 `Π[k]` 表示 Π 的长度为 k 的**初始段**
- 用 `In(Π) = Th(W ∪ {cons(δ) ∣ δ 在 Π 中出现})` 表示实施 Π 中的规则而得到的**结论集合**
- 用 `Out(Π) = {¬ψ ∣ ψ ∈ just(δ), δ 在 Π 中出现}` 表示在实施 Π 中的规则后**不能为真的公式集合**

### 定义7.8（缺省规则的可应用性）

缺省规则 `δ = φ : ψ1, ..., ψn / χ` **可应用于**一组命题集合 S，当且仅当：

```
φ ∈ S，而且 ¬ψ1 ∉ S, ..., ¬ψn ∉ S
```

### 定义7.9（过程）

设 `T = ⟨W, D⟩` 是一个缺省理论，`Π = (δ0, δ1, ...)` 是 T 的一个缺省规则序列。

- Π 是 T 的一个**过程** (Process)，当且仅当对于 Π 中的每条规则 δk, k ≥ 0，可以把 δk 应用于 `In(Π[k])`
- Π 是**成功的** (Successful)，当且仅当 `In(Π) ∩ Out(Π) = ∅`
- Π 是**封闭的** (Closed)，当且仅当对于任意 `δ ∈ D`，如果它可以被应用于 `In(Π)`，则它在 Π 中

### 例7.13（判断一个缺省规则序列是否为过程）

设 `T8 = ⟨W8, D8⟩`，`D8 = {δ1, δ2}`，`W8 = {a}`，其中：

```
δ1 = a : ¬b / ¬b
δ2 = b : c / c
```

**取 Π1 = (δ1)**：

- `In(Π1) = Th({a, ¬b})`
- `Out(Π1) = {b}`
- 由于可把 δ1 应用于 `In(Π1[0]) = In(()) = Th(W8) = {a}`（因为 `a ∈ Th({a})` 且 `¬(¬b) = b ∉ Th({a})`），所以 **Π1 是 T8 的一个过程**

**取 Π2 = (δ2, δ1)**：

- `In(Π2) = Th({a, c, ¬b})`
- `Out(Π2) = {¬c, b}`
- 由于不能把 δ2 应用于 `In(Π1[0]) = In(()) = Th({a})`（因为 b 不在其中），所以 **Π2 不是 T8 的一个过程**

### 例7.14（成功过程与封闭过程）

设 `T9 = ⟨W9, D9⟩`，`D9 = {δ1, δ2}`，`W9 = {a}`，其中：

```
δ1 = a : ¬b / d
δ2 = ⊤ : c / b
```

- **Π1 = (δ1)** 是**成功的**（因为 `In(Π1) ∩ Out(Π1) = {b} ∩ {b} = ∅`...等等，让我重新计算：`In(Π1) = Th({a, d})`，`Out(Π1) = {b}`，交为空），但**不是封闭的**，因为 δ2 可以被应用于 `In(Π1) = Th({a, d})`（δ2 的前提是 ⊤，前提满足；缺省条件 c 的否定 ¬c 不在 `Th({a, d})` 中）

- **Π2 = (δ1, δ2)** 是**封闭的**（所有可应用的规则都在序列中），但**不是成功的**，因为 `In(Π2) = Th({a, d, b})` 和 `Out(Π2) = {b, ¬c}` 都包含 b，即 `In(Π2) ∩ Out(Π2) ≠ ∅`

- **Π3 = (δ2)** 是**成功的**且**封闭的**过程

### 定义7.10（外延的操作定义）

设 `T = ⟨W, D⟩` 是一个缺省理论。一组公式集合 E 是 T 的一个**外延**，当且仅当有一个成功且封闭的 T 的过程 Π，使得 `E = In(Π)`。

### 定义7.11（过程树）

设 `T = ⟨W, D⟩` 是一个缺省理论。T 的**过程树** (Process Tree) 定义如下：

- 树的每个节点标记 In 命题集合和 Out 命题集合
  - **根节点**的 In 命题集合为 `Th(W)`，Out 命题集合为空集
  - 其他非根节点的 In 命题集合和 Out 命题集合取决于从根节点到该非根节点所应用的缺省规则
- 设 `Π[k] = (δ0, ..., δk)` 为过程树中从根节点到某个非根节点的长度为 k 的路径，则该非根节点的 In 命题集合为 `In(Π[k])`，Out 命题集合为 `Out(Π[k])`

从根节点到每个叶节点的路径对应于 T 的一个过程。对于一个过程，如果其中存在一个节点，它的 In 命题集合和 Out 命题集合的交集为非空，那么该过程是**失败**的。

## 7.4 怀疑的结论或轻信的结论

当一个缺省理论有多个外延时，处于不同外延中的命题可能存在矛盾。那么，在这种情况下可以得出什么样的结论呢？

通常，有两种推理选择：

1. **"勇敢的"或"轻信的"推理 (Credulous Reasoning)**：如果一个命题在**某个**外延中，那么可以认为该命题成立
2. **"谨慎的"或"怀疑的"推理 (Skeptical Reasoning)**：只有当一个命题在**所有**外延中，才可以认为该命题成立

形式上，对于缺省理论 `T = ⟨W, D⟩`，用 `ext(T)` 表示 T 的所有外延集合。

### 定义7.12（缺省理论的怀疑推出与轻信推出）

设 `T = ⟨W, D⟩` 是一个缺省理论，φ 为一阶命题。那么：

- T **怀疑推出** (Skeptically Entails) φ，当且仅当对于 T 的**所有**外延，φ 都在该外延中，即：

```
φ ∈ ∩[E ∈ ext(T)] E
```

- T **轻信推出** (Credulously Entails) φ，当且仅当存在 T 的**某个**外延，使得 φ 在该外延中，即：

```
φ ∈ ∪[E ∈ ext(T)] E
```

### 核心概念理解：怀疑 vs. 轻信

| 推理类型 | 态度 | 含义 |
|----------|------|------|
| **怀疑推理 (Skeptical)** | 谨慎 | 只接受在所有可能"世界"（外延）中都为真的结论 |
| **轻信推理 (Credulous)** | 大胆 | 只要有一个可能世界支持结论，就暂时接受 |

- 怀疑推理更**保守**，适合需要确定性的场景
- 轻信推理更**开放**，适合探索性场景

### 例7.16（缺省理论的怀疑推出与轻信推出）

对于例7.6中的 `T2 = ⟨W2, D2⟩`（关于团团是熊还是熊猫的例子），两个外延分别为：

- `E1 = Th(W2 ∪ {meat(tuantuan)})`
- `E2 = Th(W2 ∪ {¬meat(tuantuan)})`

可知：

- T2 **怀疑推出**：`bear(tuantuan)` 和 `panda(tuantuan)`（这两个事实在所有外延中）
- T2 **轻信推出**：`meat(tuantuan)` 和 `¬meat(tuantuan)`（分别在某一个外延中）

注意：T2 并不怀疑推出 `meat(tuantuan)` 或 `¬meat(tuantuan)`，因为它们并不在所有外延中。

### 例7.17（应用案例：AI中的推理策略）

- **怀疑推理**：适合法律和安全领域 —— 只有在结论毫无疑问时才接受（避免误判）
- **轻信推理**：适合诊断与探索性任务 —— 系统可以先接受某种假设，再通过后续证据修正

## 7.5 小结

本讲介绍了人工智能中处理不完全知识推理的重要方法——缺省逻辑。作为非单调逻辑的典型代表，它突破了经典逻辑"知识一旦成立就永远成立"的单调性限制，显著提升了系统在现实场景中的适应能力，使其能够更灵活地应对信息缺失与例外情况。

在表达形式上，缺省逻辑采用三段式的缺省规则结构，即由前提、正当性条件与结论构成的三元组，有效建模了"在没有相反证据时可接受"的常识推理。

在形式系统层面，我们系统介绍了其**语法定义**与两种**语义解释方式**：

1. **外延的不动点定义**：通过算子 Γ 的不动点刻画外延。一个句子集合 E 是外延，当 Γ(E) = E。这种定义方式清晰但非建构性。
2. **外延的操作定义**：通过缺省规则的过程树来建构性地定义外延。一个句子集合 E 是外延，当存在一个成功且封闭的过程 Π 使得 E = In(Π)。

通过对**怀疑型**与**轻信型**结论的比较，我们进一步认识到非单调系统在处理多种可能性推理时的策略差异：

- 怀疑推理：结论在所有外延中成立（∩）- 保守稳健
- 轻信推理：结论在某个外延中成立（∪）- 大胆开放

这些特性为不确定情境下的AI决策与认知建模提供了坚实的逻辑基础。

缺省逻辑不仅是非单调逻辑体系中的重要组成部分，也为后续论辩逻辑的建立奠定了方法论基础。它通过扩展一阶逻辑的表达与推理机制，为形式化常识、矛盾与例外等人类认知中普遍存在的现象提供了工具支持。同时，缺省逻辑所体现的"知识随证据更新而调整"的特性，也为哲学中的认识论和理性研究提供了形式化的启示，使我们能够更精确地分析常识性信念与不完全知识下的推理方式。

下一讲将以此为基础，进入论辩逻辑的世界，系统讨论如何刻画论证之间的攻击、防御与接受关系，从而实现对冲突性与不一致性信息的动态建模与解释性推理。

## 学习要点与AI关联

### 缺省规则

允许在信息不完整时做出合理假设，是缺省逻辑的核心工具。它体现了非单调性：新证据可能撤销旧结论。

### 缺省理论

由背景知识和缺省规则构成，能够形式化表示常识性知识。语法层面为推理提供形式化起点，便于严格定义语义与算法。其结构为 `T = ⟨W, D⟩`，W 为事实集合，D 为缺省规则集合。

### 外延语义

缺省逻辑的推理结果是一个自洽的结论集合，称为外延。同一知识库可能存在多个外延，体现了非单调推理的多解性。

外延有三种刻画方式：

- 不动点定义：E = Γ(E)
- 准归纳定义：E = ∪[i=0 to ∞] Ei（构造性）
- 操作定义：存在成功且封闭的过程 Π（建构性）

### 怀疑与轻信结论

- 怀疑推理要求结论在所有外延中成立，强调稳健性
- 轻信推理只需在某个外延中成立，体现可能性

它们为不同场景下的推理需求提供了选择。

### AI应用价值

缺省逻辑为专家系统、常识推理、知识库更新、法律推理、医学诊断等任务提供了逻辑支撑，帮助系统在不完全信息下仍能进行合理判断。

### 哲学与方法论启示

缺省逻辑刻画了"在信息不全时如何合理假设"，回应了常识推理、信念修正与理性有限性等哲学问题。它展示了逻辑如何超越演绎的封闭性，为人类认知中的例外、修正与动态知识更新提供了可计算模型。

1. 王元元. 《计算机科学中的现代逻辑学》— 以计算机科学为背景，系统介绍了缺省逻辑的基本原理和定义
2. Brewka, G. *Nonmonotonic Reasoning: Logical Foundations of Commonsense* — 总结了早期非单调逻辑的发展成果，重点介绍了缺省逻辑，并探讨了其与自认识逻辑等其他非单调逻辑系统之间的关系
3. Besnard, P. *An Introduction to Default Logic* — 一部专门围绕缺省逻辑展开的研究性著作，不仅系统介绍了其发展背景与基本概念，还深入探讨了缺省逻辑的系统性质、各类片段、证明论以及可判定性等关键问题

## 练习题

下文列出教材中的练习题（仅为知识回顾，不做解答）：

1. 单选题：以下哪种情况最适合用缺省规则建模？
2. 单选题：封闭世界假设陈述了什么？
3. 单选题：准归纳法在计算缺省理论的外延时有什么作用？
4. 单选题：为什么缺省逻辑的语义计算具有很高的计算复杂性？
5. 单选题：一个缺省理论可能有多个稳定外延，这是为什么？
6. 简答题：判断给定缺省规则是否能被应用于指定集合
7. 简答题：给定缺省理论，判断缺省规则序列是否为过程，以及是否成功和封闭
8. 简答题：用外延的不动点定义求出所有外延，并用过程树对求解过程加以说明
9. 简答题：用外延的不动点定义求缺省理论的外延（关于运动员和受伤的推理）
10. 简答题：把自然语言语句翻译为一阶缺省理论，并检查结论是否可以被轻信地或怀疑地得出（关于农户电商销售的推理）
