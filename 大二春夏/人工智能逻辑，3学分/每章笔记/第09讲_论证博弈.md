# 第9讲 论证博弈

---

## 概述

在抽象论证理论中，一个论证的地位（是否被接受、是否被拒绝、是否悬而未决）可以通过基于博弈的方法来判定。这种方法的核心理念是将论证的证明过程建模为一个两人博弈：**PRO**（提议方，Proponent）试图为某个论证提供辩护，而 **OPP**（反对方，Opponent）则试图攻击它。通过分析双方在博弈中的合法移动和胜负条件，我们可以为抽象论证框架中的论证提供一种动态的、过程导向的语义刻画。

相比于静态的语义定义（如第8讲介绍的完备语义、基语义、优先语义等），论证博弈具有以下显著优势：

- **可解释性强**：博弈的过程本身就是对论证地位的证明，其树形结构直观地展示了论证之间的攻击与防御关系
- **建构性**：博弈从一个论证出发，按照明确的规则逐步展开，得到的是一个构造性的证明，而非不动点式的定义
- **灵活性**：通过调整合法移动函数，可以适配不同的论辩语义，实现从怀疑推理到轻信推理的平滑切换

本章将系统介绍论证博弈的基本概念、赢策略的形式定义，以及在不同论辩语义下的合法移动函数设计。

---

## 9.1 争辩树和赢策略

论证博弈的起点是一个抽象论证框架 `AF = ⟨AR, attacks⟩`，其中 `AR` 是论证的集合，`attacks ⊆ AR × AR` 是论证之间的攻击关系。对于任何一个论证 `α ∈ AR`，我们可以从它出发，沿着攻击关系的逆方向展开一棵树，表示所有可能的博弈路径。

### 定义9.1（争辩树，Dispute Tree）

给定一个抽象论证框架 `AF = ⟨AR, attacks⟩` 和一个论证 `α ∈ AR`，由 `α` 诱导的**争辩树** `T` 是一棵以论证为节点标签的树，满足以下条件：

1. 根节点为 `α`
2. 在 `T` 中，`β` 是 `γ` 的子节点，当且仅当 `(β, γ) ∈ attacks`，即 `β` 攻击 `γ`

换言之，争辩树从目标论证出发，逐层向上（向攻击者方向）展开。每一层对应博弈的一个回合：奇数次移动由 OPP 执行（攻击当前论证），偶数次移动由 PRO 执行（攻击 OPP 的论证以进行防御）。

---

#### 核心概念理解：争辩树的结构

- **根节点**：需要证明的目标论证 `α`，由 PRO 提出
- **第一层（子节点）**：所有攻击 `α` 的论证，由 OPP 提出
- **第二层（孙节点）**：所有攻击第一层论证的论证，由 PRO 提出作为防御
- **以此类推**：博弈交替进行，直到一方无法继续移动

每一棵争辩树对应了所有可能的博弈路径，不同的路径反映了 OPP 可能选择的不同攻击策略。

---

### 例9.1（带有循环的争辩树）

考虑如下抽象论证框架 `AF₁`：

```
AR = {α, β, γ}
attacks = {(β, α), (γ, β), (α, γ)}
```

攻击关系形成了一个三元环：`α → γ → β → α`。

从论证 `α` 出发的争辩树如下（使用下标区分不同位置的同一论证）：

```
                    α
                    |
                    β        （OPP: β 攻击 α）
                    |
                   γ₁        （PRO: γ 攻击 β）
                    |
                   α₁        （OPP: α 攻击 γ）
                    |
                   β₁        （PRO: β 攻击 α）
                    |
                   γ₂        （OPP: γ 攻击 β）
                    |
                   α₂        （PRO: α 攻击 γ）
                    |
                   ...       （无限展开）
```

由于存在攻击循环，这棵争辩树是**无限的**（infinitely wide or deep）。这说明，在存在循环的论证框架中，争辩树可能是无限树，需要引入修剪规则来获得有限的证明。

---

### 定义9.2（赢策略，Winning Strategy）

在争辩树中，**一个争辩（dispute）** 是从根节点到某个叶节点的一条路径。一次博弈的胜负判定如下：

- 如果一方完成了最后一次合法移动，而另一方无法做出任何回应，则做出最后一次移动的一方**赢得**这次争辩
- 一条**辩护线（defence line）** 是 `α` 的一个争辩，在该争辩中 PRO 赢得了博弈（即 PRO 完成了最后一次移动）

对于目标论证 `α`，PRO 在争辩树中拥有**赢策略（winning strategy）**，当且仅当==在争辩树中的**所有**争辩（即所有可能的博弈路径）中，PRO 都赢得了博弈。==

==换句话说，无论 OPP 选择哪种攻击路径，PRO 都能够做出有效回应并最终获胜==。

---

#### 核心概念理解：赢策略的直观含义

- **PRO 赢一次争辩**：在此次博弈中，PRO 成功回应了 OPP 的所有攻击
- **PRO 拥有赢策略**：PRO 能够应对 OPP 的**任何**可能的攻击——这意味着 PRO 有一个系统性的防御方案
- ==**OPP 拥有赢策略**：存在至少一条攻击路径，使得 PRO 无法做出有效回应==

如果 PRO 对 `α` 拥有赢策略，则说明 `α` 在论证框架中是可辩护的——这正是论证博弈与论证语义之间的核心联系。

---

### 例9.2（辩护线和赢策略的示例）

考虑抽象论证框架 `AF₂`：

```
AR = {α, β₁, β₂, γ}
attacks = {(β₁, α), (β₂, α), (γ, β₁)}
```

从 `α` 出发的争辩树为：

```
               α
              / \
             /   \
            β₁    β₂     （OPP的两种攻击选择）
            |
            γ             （PRO对β₁的防御）
```

分析两个争辩路径：

1. **路径 `α - β₁ - γ`**：OPP 攻击 `α`（提出 `β₁`），PRO 防御（提出 `γ` 攻击 `β₁`），OPP 无法回应（没有论证攻击 `γ`），PRO 获胜
2. **路径 `α - β₂`**：OPP 攻击 `α`（提出 `β₂`），PRO 无法回应（没有论证攻击 `β₂`），OPP 获胜

由于存在一条 PRO 失败的路径（路径2），因此 PRO **不具有**对 `α` 的赢策略。路径1虽然 PRO 赢了，但这只是一条辩护线，不足以构成赢策略。

---

再考虑抽象论证框架 `AF₃`：

```
AR = {α, β₁, β₂, γ₁, γ₂}
attacks = {(β₁, α), (β₂, α), (γ₁, β₁), (γ₂, β₂)}
```

从 `α` 出发的争辩树为：

```
               α
              / \
             /   \
            β₁    β₂     （OPP的两种攻击选择）
            |      |
            γ₁     γ₂    （PRO对两种攻击的防御）
```

分析所有争辩路径：

1. **路径 `α - β₁ - γ₁`**：PRO 获胜（OPP 无法回应 `γ₁`）
2. **路径 `α - β₂ - γ₂`**：PRO 获胜（OPP 无法回应 `γ₂`）

在**所有**可能的路径中 PRO 都获胜，因此 PRO **具有**对 `α` 的赢策略。这表明 `α` 是可辩护的。

---

### 无限争辩与剪枝规则

在存在循环的论证框架中（如例9.1），争辩树可能是无限的。为了获得有限的博弈分析，需要引入剪枝规则来截断由重复论证引起的无限路径。

一种常见的剪枝策略是：==**禁止在同一个争辩路径中重复使用论证**。如果在当前争辩中，某个论证已经出现过，则该论证不能再被任何一方使用==。这一规则反映了论证博弈的基本直觉：在同一个论证链条中，如果一个论证需要依赖自身来获得辩护，这种循环辩护是不合法的。

---

### 定义9.3（子争辩与赢策略的形式化定义）

设 `T` 是一棵争辩树，`D_T` 是 `T` 中所有争辩（从根到叶的路径）的集合。`T` 是 `α` 的一个**赢策略**，当且仅当以下三个条件同时满足：

1. **非空有限性**：`D_T` 是一个非空的、有限的、由有限争辩组成的集合，且每条争辩都以 PRO 获胜结束
2. **全面覆盖性**：对于每一条争辩 `d ∈ D_T`，以及 `d` 中以 PRO 的论证 `β` 结束的每一个子争辩 `d'`，都有：对于每一个攻击 `β` 的论证 `γ`，存在另一条争辩 `d'' ∈ D_T`，使得 `d' - γ_opp` 是 `d''` 的一个子争辩

其中，`d' - γ_opp` 表示在子争辩 `d'` 之后，OPP 选择用 `γ` 来攻击 `β`（即延长了一条边 `(γ, β)`）。

条件2的直观含义是：对于 PRO 的每一个防御性论证 `β`，PRO 必须能够应对 OPP 对 `β` 的**每一种**可能的攻击。这体现了赢策略的核心要求——==PRO 必须对所有可能的攻击路径都有所准备。==

---

### 例9.3（法律推理案例中的赢策略）

假设在一个法律论证中，有以下抽象论证框架：

```
AR = {α, β₁, β₂, γ₁₁, γ₁₂, γ₂₁}
attacks = {
    (β₁, α),   // 对方提出的第一个反驳
    (β₂, α),   // 对方提出的第二个反驳
    (γ₁₁, β₁), // PRO对第一个反驳的第一次辩护
    (γ₁₂, β₁), // PRO对第一个反驳的第二次辩护（备选）
    (γ₂₁, β₂)  // PRO对第二个反驳的辩护
}
```

从 `α` 出发的争辩树可以表示为：

```
                        α
                       / \
                      /   \
                    β₁     β₂
                   / \      |
                  /   \     |
               γ₁₁   γ₁₂   γ₂₁
```

对应的争辩集合：

- `d₁: α - β₁ - γ₁₁`（PRO 获胜）
- `d₂: α - β₁ - γ₁₂`（PRO 获胜）
- `d₃: α - β₂ - γ₂₁`（PRO 获胜）

对于子争辩 `d' = α - β₁`（以 OPP 的论证 `β₁` 结束），这里 PRO 用 `γ₁₁` 或 `γ₁₂` 回应，但此子争辩以 OPP 的论证结束，所以条件2不直接适用于此。

对于每个以 PRO 的论证结束的子争辩（如 `d₁` 中以 `γ₁₁` 结束），需要检查 OPP 对 `γ₁₁` 的所有可能攻击是否都能被 PRO 覆盖。如果没有任何论证攻击 `γ₁₁`，则此处的条件自动满足。

由于每一条争辩都以 PRO 获胜结束，且条件2也满足（此处每个 PRO 的终结论证都没有攻击者），因此这棵争辩树构成了 `α` 的一个赢策略。

## 9.2 各种论辩语义下合法移动函数

不同的论辩语义对论证的接受条件有不同的要求，因此博弈中允许的移动规则也需要相应地调整。本节引入**合法移动函数**的概念，通过定义不同的移动约束来适配基语义和优先语义。

### 定义9.4（合法移动函数，Legal Move Function）

**合法移动函数** `φ` 是一个从争辩集合到论证集合的映射：

```
φ: D_T → 2^AR
```

其中 `D_T` 是争辩树 `T` 中所有子争辩的集合。对于任意一个子争辩 `d ∈ D_T`，`φ(d)` 给出了在当前博弈状态下允许被提出的论证集合。

---

#### 核心概念理解：合法移动函数的作用

合法移动函数 `φ` 在两个方面发挥作用：

1. **修剪无限树**：通过对移动内容施加约束，将无限的争辩树修剪为有限的 **φ 争辩树（φ-tree）**
2. **适配语义**：不同的论辩语义对应不同的合法移动函数，使得博弈的胜负结果与语义的接受判定保持一致

例如，"禁止重复论证"这一规则就可以通过合法移动函数来实现：如果一个论证已经出现在子争辩 `d` 中，则 `φ(d)` 中不包含该论证。

---

### 定义9.5（φ争辩树与φ赢策略）

给定一棵争辩树 `T` 和一个合法移动函数 `φ`，一棵 **φ争辩树（φ-tree）** 是通过根据 `φ` 修剪原始树 `T` 而获得的子树。具体地，在 φ争辩树中，节点 `β` 是节点 `γ` 的子节点（即从 `γ` 可以移动到 `β`），当且仅当：

- `(β, γ) ∈ attacks`（β 攻击 γ），且
- `β ∈ φ(d)`，其中 `d` 是从根节点到 `γ` 的子争辩

**φ赢策略** 的定义与定义9.3一致，只需将 "争辩树" 替换为 "φ争辩树"。

---

### 例9.4（不同φ下的争辩树和赢策略）

回顾例9.1中的抽象论证框架 `AF₁ = ⟨{α, β, γ}, {(β, α), (γ, β), (α, γ)}⟩`。

在不同的合法移动函数下，对应的 φ争辩树和赢策略也不同：

- **若不禁止重复论证**：争辩树无限展开（如例9.1所示），无法判定输赢
- **若禁止双方重复论证**：争辩树被修剪为有限树。从 `α` 出发，第一层 `β`，第二层 `γ`，第三层 `α` 被禁止（因为 `α` 已在根节点出现过），OPP 无法继续 → PRO 在有限步内获胜

不同的剪枝策略会直接影响到赢策略的存在性和胜负判定。

---

### 博弈参与者的不同目标

在论证博弈中，PRO 和 OPP 有着截然不同的目标：

| 参与者 | 目标 | 策略性质 |
|--------|------|----------|
| **PRO** | 构建一个辩证的证明（dialectical proof），证明目标论证是可接受的 | 建设性的（constructive） |
| **OPP** | 阻止 PRO 完成证明，找出 PRO 论证中的弱点 | 破坏性的（destructive） |

**举证责任（Burden of Proof）** 在不同推理模式下有所不同：

- **怀疑推理（Skeptical Reasoning）**：PRO 的举证责任更重。要证明一个论证在怀疑推理下可接受，PRO 必须以绝对的优势赢得博弈
- **轻信推理（Credulous Reasoning）**：OPP 的举证责任更重。要阻止 PRO 证明一个论证，OPP 必须找到 PRO 无法防御的攻击路径

这种不对称的举证责任分配通过合法移动函数中不同的约束条件来实现。

---

### （1）基语义下的合法移动函数

在基语义（Grounded Semantics）下，要证明论证 `α` 处于基外延中，必须能够在不借助 `α` 本身的情况下为 `α` 辩护。换言之，PRO ==不能使用循环论证（circular argumentation）。==

---

#### 定义9.6（基语义下的合法移动函数 φ_gr）

对于任意子争辩 `d`，基语义下的合法移动函数 `φ_gr` 定义如下：

设当前需要被回应的论证为 `α`（即上一次移动的论证）。

- **若 `|d|` 为奇数（OPP 的回合）**：
  ```
  φ_gr(d) = {β | (β, α) ∈ attacks}
  ```
  即 OPP 可以提出任何攻击 `α` 的论证

- **若 `|d|` 为偶数（PRO 的回合）**：
  ```
  φ_gr(d) = {β | (β, α) ∈ attacks, β ∉ PRO(d)}
  ```
  即 PRO 可以提出任何攻击 `α` 的论证，==但**不能重复使用自己之前在该争辩中已经提出过的论证**==

其中，`PRO(d)` 表示在子争辩 `d` 中 PRO 已经提出的论证集合。

---

#### 核心概念理解：φ_gr 的设计动机

基语义对应的是**最小的不动点**——论证需要从"底层"（无攻击者的论证）逐步向上构建。因此：

- ==PRO 禁止重复自身使用的论证，==这防止了循环辩护（即用 `β` 来防御 `α`，而 `β` 本身又依赖于 `α` 或其他尚未得到独立辩护的论证）
- ==OPP 则没有此限制——OPP 可以自由地使用任何攻击手段==

这一设计体现了基语义的怀疑推理特性：PRO 必须为论证==提供无循环的、严格递增的辩护链。==

---

#### 性质（基语义的博弈刻画）

对于任何有限的抽象论证框架 `AF`，如下两个命题是等价的：

- 论证 `α` 属于 `AF` 的基外延
- 存在 `α` 的一个 `φ_gr`-赢策略 `T`，且 `PRO(T)` 是无冲突的（conflict-free）

其中 `PRO(T)` 表示在赢策略 `T` 中 PRO 使用的所有论证的集合。

---

### 例9.5（φ_gr赢策略示例）

考虑抽象论证框架 `AF₄`：

```
AR = {α, β₁, β₂, γ}
attacks = {(β₁, α), (β₂, α), (γ, β₁)}
```

我们从 `α` 出发，使用 `φ_gr` 构建争辩树：

第一回合（OPP 的回合，`|d|=1`，奇数）：OPP 可以提出 `β₁` 或 `β₂`（两个都攻击 `α`）

- 若 OPP 选择 `β₁`：
  - 第二回合（PRO 的回合，`|d|=2`，偶数）：PRO 可以提出 `γ`（攻击 `β₁`，且 `γ` 未被 PRO 使用过）
  - 第三回合（OPP 的回合，`|d|=3`，奇数）：没有论证攻击 `γ`，OPP 无法回应 → PRO 获胜

- 若 OPP 选择 `β₂`：
  - 第二回合（PRO 的回合，`|d|=2`，偶数）：没有论证攻击 `β₂`，PRO 无法回应 → OPP 获胜

由于存在一条 PRO 失败的路径，PRO 不具有对 `α` 的 `φ_gr`-赢策略。因此 `α` 不在基外延中。

---

再考虑另一个论证 `γ`。从 `γ` 出发：

- 第一回合（OPP）：没有论证攻击 `γ`，OPP 无法移动 → PRO 直接获胜

因此 `γ` 存在 `φ_gr`-赢策略，`γ` 在基外延中。

---

### （2）轻信优先语义下的合法移动函数

在优先语义（Preferred Semantics，也称轻信优先语义）下，论证只需要属于某个可相容外延（admissible extension）。此时，循环辩护在某种程度上是被允许的，因为可能存在互相防御的论证集合。

---

#### 定义9.7（轻信优先语义下的合法移动函数 φ_pr）

对于任意子争辩 `d`，轻信优先语义下的合法移动函数 `φ_pr` 定义如下：

设当前需要被回应的论证为 `α`。

- **若 `|d|` 为奇数（OPP 的回合）**：
  ```
  φ_pr(d) = {β | (β, α) ∈ attacks, β ∉ OPP(d)}
  ```
  即 OPP 可以提出任何攻击 `α` 的论证，但**不能重复使用自己之前在该争辩中已经提出过的论证**

- **若 `|d|` 为偶数（PRO 的回合）**：
  ```
  φ_pr(d) = {β | (β, α) ∈ attacks, β ∉ OPP(d), (β, β) ∉ attacks}
  ```
  即 PRO 可以提出任何攻击 `α` 的论证，但需满足：
  1. 该论证没有被 OPP 在该争辩中使用过
  2. 该论证不自攻（即不存在 `(β, β) ∈ attacks`）

其中，`OPP(d)` 表示在子争辩 `d` 中 OPP 已经提出的论证集合。

---

#### 核心概念理解：φ_pr 的设计动机

`φ_pr` 与 `φ_gr` 的关键区别在于举证责任的分配：

| 约束条件 | φ_gr（基语义） | φ_pr（优先语义） |
|----------|---------------|-----------------|
| OPP 禁止重复自身论证 | 否 | **是** |
| PRO 禁止重复自身论证 | **是** | 否 |
| 禁止自攻论证（PRO） | 否 | **是** |

- **OPP 禁止重复**：由于优先语义采用轻信推理，OPP 的举证责任更重。OPP 不能无限循环地使用同一个论证来攻击 PRO——如果 OPP 重复，说明 OPP 的论证本身也面临同样的辩护问题
- **PRO 不受重复限制**：PRO 可以循环辩护，因为轻信推理下允许一组互相防御的论证共同被接受
- **禁止自攻论证**：自攻的论证在所有论辩语义下都是不可接受的，因此 PRO 不能使用它们

---

#### 性质（优先语义的博弈刻画）

对于任何有限的抽象论证框架 `AF`，如下两个命题是等价的：

- 论证 `α` 属于 `AF` 的某个可相容外延（admissible extension），从而也属于某个优先外延（preferred extension）
- 存在 `α` 的一个 `φ_pr`-赢策略 `T`，且 `PRO(T)` 是无冲突的

---

### 例9.6（φ_pr赢策略示例）

考虑抽象论证框架 `AF₅`：

```
AR = {α, β, γ}
attacks = {(β, α), (γ, β), (α, γ)}
```

这是一个三元循环结构：`α` 攻击 `γ`，`γ` 攻击 `β`，`β` 攻击 `α`。

**在基语义下**：没有不被攻击的论证，因此基外延为空，`α` 不被接受

**在优先语义下**：

从 `α` 出发，使用 `φ_pr` 构建争辩树：

- 第一回合（OPP，`|d|=1`）：OPP 提出 `β`（攻击 `α`），`OPP(d) = {β}`
- 第二回合（PRO，`|d|=2`）：PRO 不能提出 `β`（OPP 已用过），`γ` 攻击 `β` 且不自攻 → PRO 提出 `γ`
- 第三回合（OPP，`|d|=3`）：OPP 不能提出 `γ`（OPP 已用过... 等等，`γ` 是 PRO 提出的，OPP 检查 `OPP(d)`，之前只用了 `β`），所以 OPP 可以提出 `α`（攻击 `γ`）
- 第四回合（PRO，`|d|=4`）：PRO 检查：`β` 攻击 `α`，`β` 不在 `OPP(d) = {β}` 中？实际上 `β` 在 OPP(d) 中，所以 PRO 不能使用 `β`！由于没有其他论证攻击 `α`，PRO 无法回应 → PRO 失败

等等，这里需要仔细分析。在 `φ_pr` 下，PRO 禁止使用 OPP 已用过的论证。如果 OPP 在第三回合使用了 `α`，而 PRO 想用 `β` 来攻击 `α`，但 `β` 已经被 OPP 用过，PRO 就无法使用。PRO 失败。

但这与优先语义的预期不符——在优先语义下，`{α, β, γ}` 可能构成一个可相容集。问题在于，这里的博弈是对单条争辩路径的分析。

实际上，对于循环的论证框架 `AF₅`，由于 `φ_pr` 禁止 OPP 重复论证，OPP 只能进行有限步。如果 OPP 在第三回合不能提出 `α`（因为？`α` 不在 OPP(d) 中：OPP(d) = {β}，所以 `α` 不在其中，OPP 可以提出），然后在第四回合 PRO 无法使用 `β`（因为 `β` 在 OPP(d) 中），所以 PRO 失败。

但是，在另一条路径上，如果 OPP 无法提出新的论证（因为所有攻击者都已被 OPP 使用过），PRO 仍然获胜。

结论：对于 `AF₅`，在 `φ_pr` 下 PRO 是否拥有赢策略取决于对重复移动的具体处理方式。教材中通常按以下方式分析：由于存在攻击循环，PRO 和 OPP 都会用尽可用的论证，最终根据禁止规则决定胜负。

## 9.3 小结

本讲从论证博弈的视角重新审视了抽象论证理论中的核心问题：一个论证何时被认为是可接受的？答案不是通过静态的不动点方程，而是在一个动态的双人博弈过程中逐步构建出来的。

我们首先介绍了**争辩树**（Dispute Tree）的概念——从目标论证出发，沿攻击关系的逆方向展开一棵树，表示所有可能的博弈路径。每一条从根到叶的路径对应一个具体的博弈过程（争辩）。如果 PRO 在所有可能的博弈路径中都能获胜，则 PRO 拥有对该论证的**赢策略**（Winning Strategy）。

赢策略的形式化定义（定义9.3）要求 PRO 能够：
1. 在每一条争辩中都能做出最后的有效移动
2. 对于任何一个防御性论证，都能够应对 OPP 对其的所有可能的攻击

随后，我们引入了**合法移动函数** `φ` 的概念。不同的论辩语义对 PRO 和 OPP 的移动有不同的约束条件，这些约束通过 `φ` 来形式化：

- **基语义**（`φ_gr`）：PRO 禁止重复使用自身论证，体现怀疑推理——论证必须严格地从底层向上构建，不能有循环辩护
- **优先语义**（`φ_pr`）：OPP 禁止重复使用自身论证，PRO 可以使用 OPP 未用过的任何论证（除自攻论证外），体现轻信推理——允许互相防御的论证集合共同被接受

两种合法移动函数的设计反映了不同的举证责任分配：基语义下 PRO 的举证责任更重（必须提供无循环的辩护链），而优先语义下 OPP 的举证责任更重（必须找到 PRO 无法防御的攻击）。

论证博弈的核心贡献在于：将静态的论辩语义转化为动态的、过程导向的推理机制。这种机制具有极高的可解释性——博弈的每一步都是透明的，用户可以追踪论证的全部辩护链。此外，通过灵活配置合法移动函数，同一个论证框架可以支持从保守的怀疑推理到开放的轻信推理的平滑切换，为不同场景下的AI推理提供了坚实的形式化工具。

下一讲将进一步探讨论证框架的扩展，包括结构化论证（structured argumentation）和基于偏好的论证（preference-based argumentation），将抽象论证的理论扩展到更丰富的表达层面。

## 学习要点与AI关联

### 争辩树（Dispute Tree）

从目标论证出发，沿攻击关系逆向展开的树形结构。树根是待证论证，每条边表示一次攻击。争辩树的每一条路径对应一次具体的博弈过程。循环攻击关系会导致无限争辩树。

### 赢策略（Winning Strategy）

PRO 拥有赢策略意味着：无论 OPP 如何选择攻击路径，PRO 都有能力做出有效回应并最终获胜。这是博弈方法对论证接受性的刻画方式。形式化定义要求：所有争辩都以 PRO 获胜结束，且 PRO 的每个防御性论证都能抵抗 OPP 的所有可能攻击。

### 合法移动函数（Legal Move Function）

`φ: D_T → 2^AR` 定义了在当前博弈状态下允许被提出的论证集合。通过不同的 `φ`，可以实现不同的语义：

- `φ_gr`（基语义）：PRO 禁止重复——无循环辩护
- `φ_pr`（优先语义）：OPP 禁止重复——允许循环辩护

合法移动函数是连接博弈规则与论辩语义的桥梁。

### 怀疑推理 vs. 轻信推理在博弈中的体现

| 推理模式 | 语义 | 举证责任 | 博弈特性 |
|----------|------|---------|---------|
| 怀疑推理 | 基语义 | PRO 更重 | PRO 需提供严格无循环的辩护 |
| 轻信推理 | 优先语义 | OPP 更重 | OPP 不能重复论证，PRO 可以循环辩护 |

### AI应用价值

论证博弈为以下AI任务提供了理论支撑：

- **自动辩论系统**：模拟人类辩论过程，评估论证力度
- **可解释AI（XAI）**：通过博弈树可视化，使AI的推理过程对用户透明
- **多Agent对话系统**：在多Agent环境中协调冲突的论证
- **法律推理辅助**：帮助分析法律论证中的攻击与防御关系
- **决策支持系统**：评估支持和反对某个决策的论证力量对比

### 哲学与方法论启示

论证博弈将"真理"的确定从静态的语义指派转变为一个动态的辩证过程。这与哲学中苏格拉底式的辩证法（dialectic）有着深刻的呼应——真理不是在独白中发现的，而是在对话和辩驳中逐渐显现的。基于博弈的论证理论为这种辩证推理提供了精确的形式化模型。
