# 第13讲 因果推理

## 概述

在新时代伟大变革背景下，从科教兴国到乡村振兴，从生态文明建设到高质量发展，无不蕴含着深刻的因果关系。而因果推理理论正是研究从数据或案例中发现因果规律、进行因果预测的推理过程。从本质上看，因果推理研究的是"原因"与"结果"之间的一种结构关联。

因果推理是人类认识世界、改造世界的重要思维方式，是贯穿哲学社会科学诸多学科领域的基础性概念和推理方法。深入系统地理解因果推理理论，不仅有助于提升逻辑思维和问题分析能力，也对于建设中国自主知识体系具有重要的理论与实践意义。本讲从结构因果模型理论出发，将因果推理理论与中国现代化进程中涌现出的丰富实践经验紧密结合。

## 13.1 什么是因果推理

在最基本的层面上，因果推断是一门规范化识别、建模和量化因果关系的学科。然而，关于什么是因果关系，却难以清晰定义。

### 休谟的因果定义

十八世纪的休谟从现象与现象的联系中给出了有关因果关系的定义：

1. 如果A发生，则B必然发生；
2. 如果A不发生，则B也不发生；
3. A发生在B之前。此时，称A是B的原因。

尽管上述三个条件符合对因果关系的常识理解，但仍然可以推出"公鸡打鸣是太阳升起的原因"这样的荒谬结论。这一观察说明，**因果性与关联性有密切联系，但存在本质区别**。

### 核心概念理解：相关性 vs 因果性

**相关性（Correlation）**：两个事件同时出现，但可能并无直接联系。例如：公鸡打鸣和太阳升起高度相关，但前者并不导致后者。

**因果性（Causation）**：一个事件的发生会改变另一个事件的发生概率。例如：吸烟显著增加患肺癌的概率。

因果推理的重点是**从相关中识别因果**。依据著名学者朱迪亚·珀尔（Judea Pearl）的观点，因果的概念与统计的概念不同。前者研究变量与变量之间分布的动态特性，而统计则研究静态特性。其根本原因是，为了识别和评估变量之间的虚假关联，仅凭统计的概念是无法实现的。

### 关联性不等于因果性

给定待研究的两个变量X和Y，当它们的取值在统计意义上呈现非独立性，即：

```
P(Y | X) ≠ P(Y)
```

我们说它们存在关联。然而，关联性并不一定意味着因果性。

**例子**：自2023年起，我国西部绿色能源产业发展（T）与东部地区人工智能企业模型训练成本降低（Y）高度正相关，但二者无直接因果关系，真正的共同原因是2022年起国家正式实施了"东数西算"工程（X）。

在这里，把"国家实施东数西算工程"称为**混杂变量（Confounding Variable）**，它同时影响自变量T和因变量Y，导致观察到的T-Y关联性包含虚假因果信号。

### 干预是识别因果的关键方法

因果性与关联性存在本质区别：

- 如果两个变量X与Y之间存在**因果性**，那么通过对X施加干预（如强制改变X的值）会导致Y变化。
- 相反，如果两个变量X与Y之间仅仅存在**关联性**，而无因果性，那么通过干预X，往往无法改变Y。

例如，通过干预我国西部绿色能源产业发展并不会直接导致东部地区人工智能企业训练成本的升高，而暂停东数西算工程会直接导致东部企业训练成本的显著增加。

由上述分析可知，**通过干预来改变明确存在关联关系的某变量取值，研究变量取值改变对结果变量的影响，是识别和评估因果关系的一个重要方法**。

### 类型因果 vs 实际因果

这种方法普遍适用于关于"类型"的因果关系，比如"吸烟会导致肺癌"这样规律性的因果关系。而对于具体已经发生的事件的因果推断，如"一位长期吸烟的男性得了肺癌"这样的实际因果，则需要借助**反事实推理**的方法。

## 13.2 结构因果模型

在诸多因果模型理论中，珀尔建立的**结构因果模型（Structural Causal Model, SCM）**最受关注、最为重要。这是一种描述现实世界相关特征及其相互作用的模型，提供了两种重要的因果推理方法：**干预推理方法**和**反事实推理方法**。

### 模型的组成

结构因果模型由两个变量集合U和V，以及赋值函数集合F组成，这些函数根据模型中其他变量的值为V中的每个变量分配值。

- **集合U**：外生变量（Exogenous Variables）。一般而言，它们是模型外部的，并被假定彼此独立，通常不解释它们是如何产生的，可以理解为未被观测到的因素。
- **集合V**：内生变量（Endogenous Variables）。它们是模型内部由其他变量决定的变量。

每个结构因果模型都可以被看作一个有向图，其中外生变量和内生变量共同组成了图中的节点，而赋值函数可以被理解为有向边。直观上，如果变量X有指向变量Y的箭头，那么就表明X是Y的一个原因。

模型中的每个内生变量至少和一个外生变量组成一条有向边，并且外生变量是这条边的起点，内生变量是这条边的终点。外生变量不能在与任何其他变量组成的边中作为终点，特别是不能在与内生变量组成的边中作为终点。如果知道每个外生变量的值，那么通过使用函数集合F，能够完全地确定每个内生变量的值。

**例子**：2022年教育部印发《农村义务教育学生营养改善计划实施办法》的通知，我们想研究农村学生营养午餐计划X与农村学生学习成绩Y之间的因果关系。我们可能假设Y还依赖于当地气候条件Z。在这种情况下，我们将X和Y称为内生变量，而将Z称为外生变量。这是因为我们假设气候条件是一个外部因素，即它不能由是否实施营养早餐计划和学生成绩直接决定。

结构因果模型通过显式表示因果机制来区分不同外部干预操作的影响，从而可以描述外部世界发生的变化。

### 定义13.1（结构因果模型）

结构因果模型是一个三元组M = ⟨U, V, F⟩，其中：

- **U**是由变量{U_1, U_2, ..., U_M}组成的集合，称为外生变量，由模型外部因素决定；
- **V**是由变量{V_1, V_2, ..., V_n}组成的集合，称为内生变量，它们由模型中的变量（即U ∪ V）决定；
- **F**是由{f_1, f_2, ..., f_n}组成的函数集合，称为结构方程。每个f_i都是从变量集(U' ∪ PA_i)的所有可能取值到{V_i}的所有可能取值的一个函数，其中U' ⊆ U，PA_i ⊆ V \ {V_i}。

直观上，对于每个i，函数f_i根据导致V_i原因的外生变量U'的取值和内生变量PA_i的取值为V_i指定一个值，即：

```
V_i = f_i(PA_i, U'),   i = 1, ..., n
```

每个结构因果模型M与有向图G(M)相关联，其中每个节点对应一个变量，有向边是从(U' ∪ PA_i)中的节点指向V_i。称G(M)是结构因果模型M的**因果图**。

### 定义13.2（子模型）

令M是一个结构因果模型，X为V中的一组变量集合，x是X的一个具体赋值。M的子模型M_x为：

```
M_x = ⟨U, V, F_x⟩
```

其中F_x = {f_i : V_i ∉ X} ∪ {X = x}。换句话说，F_x是通过删除F中与集合X的变量相应的所有函数f_i，并将其替换为常量函数X = x的集合而组成的。

子模型用于表示我们假设局部因素发生变化后对整个模型产生的效应，包括那些由反事实的前提所隐含的效应。如果我们将F中的每个函数f_i分别解释为一个独立的物理机制，并将行动do(X = x)定义为在任意u下M的最小变化使X = x成立，那么M_x表示这个最小变化所对应的模型，因为M_x与M的区别仅在于M_x直接确定了X被决定的机制（X = x）。

### 例13.1（关于群众健康满意度的结构因果模型）

党的二十大报告指出，"把保障人民健康放在优先发展的战略位置，完善人民健康促进政策"，结合新形势下人工智能技术的发展，多地开始从传统医疗向智慧医疗转型。这其中，群众健康满意度（Z）就是由社区医院AI诊断设备数量（X）和医生AI工具培训时长（Y）共同决定，它们之间的决定关系大概可以量化的表示为：

```
Z = 2X + 3Y
```

涉及它们三者的结构因果模型可以符号化地如下表示：

```
U = {X, Y}, V = {Z}, F = {f_Z}
f_Z: Z = 2X + 3Y
```

## 13.3 干预与因果效应

由于待研究的变量之间相关性不等于因果性，为了分析因果关系，一种传统的方法是通过**随机对照实验（RCT）**。在该实验中，影响结果变量的输入变量只有一个是随机变化的，其他均保持不变，因此，若结果变量发生变化，则一定是由该输入变量引起的。

尽管随机对照实验在物理学、生物学、医学等领域得到了广泛的应用，但在只有观察数据而没有实验数据的领域，该方法难以被应用。

珀尔等人通过引入**do算子（do-operator）**，基于结构因果模型提出了一种全新的形式化理论，实现了一种可以从观察数据进行因果分析和推断的方法。

### （1）行为效应与操纵图模型

给定一个因果图，对其中的某个变量X进行干预并固定其值为x时，可将所有指向X的边移除，得到一个子模型。子模型可以用于表示局部行动的效用，称为**行为效应**。

#### 定义13.3（行为效应）

令M是一个结构因果模型，X为V中的一组变量集合，x是X的一个具体赋值。子模型M_x定义了行动do(X = x)对M的**行为效应**。

对变量干预之后得到的子模型所对应的因果图也称为**操纵图模型（Manipulated Graph Model）**。

### 辛普森悖论（Simpson's Paradox）

在因果推理理论研究中，辛普森悖论代表了一类重要的现象，即**混杂变量引起的整体统计关系与分组统计关系之间可能出现的矛盾**。而利用do算子和基于干预的分析方法，可以有效解释这一悖论的成因，并找到导致悖论的混杂变量。

#### 例13.2（辛普森悖论）

表13.1列出了某组患者在是否服用新药以后的恢复情况。

**表13.1：对新药的研究结果（考虑性别因素）**

| 性别 | 用药 | 不用药 |
|------|------|--------|
| 男   | 81/87 (93%) 恢复 | 234/270 (87%) 恢复 |
| 女   | 192/263 (73%) 恢复 | 55/80 (69%) 恢复 |
| 混合 | 273/350 (78%) 恢复 | 289/350 (83%) 恢复 |

当不考虑性别因素时，未用药患者的恢复率（83%）高于用药患者的恢复率（78%）。然而在按性别分组之后，男女两组中用药患者的恢复率均高于不用药患者的恢复率。这一反直觉的现象即为**辛普森悖论**。

**原因分析**：辛普森悖论的主要原因是"性别"这一变量导致了用药与否和恢复率之间的虚假关联。具体来说，在这个例子中，有三个变量：用药情况（X）、恢复率（Y）和性别（Z）。其中性别是混杂变量。

这三个变量之间的关系，以及它们与外部不确定因素(U_X, U_Y, U_Z)的关系，可以构建一个结构因果模型M = ⟨U, V, F⟩，其中：
- U = (U_X, U_Y, U_Z)
- V = {X, Y, Z}
- F由如下三个函数构成：
  - f_X = (Z, U_X)（用药情况受性别和外部因素影响）
  - f_Y = (Z, X, U_Y)（恢复率受性别、用药情况和外部因素影响）
  - f_Z = (U_Z)（性别完全由外部因素决定）

相应的因果图如**图13.1**所示：Z（性别）同时指向X（用药）和Y（恢复），X也指向Y。

为了比较患者用药与否的恢复情况，可以对患者的用药情况进行干预，即将患者随机分配至用药组与未用药组。具体而言，干预过程中固定系统中的其他变量，然后改变某一变量的取值，观察其他变量的变化。通过对X进行干预，得到子模型M_x = ⟨U, V, F_x⟩，其中F_x = {f_Y, f_Z, X = x}。

该子模型对应的操纵图如**图13.2**所示：在图中，指向X的边被移除（即Z不再影响X），X被固定为x。该操纵图模型表示固定X的值为x，其他关系和变量保持不变，进而能够通过干预X的情况下更准确地研究Z与Y之间的关系。

### （2）因果效应差

给定上述操纵图模型，用P(Y = y | do(X = x))表示对X进行干预，固定其值为x时，Y = y的概率。把P(Y = y | do(X = x))称为**因果效应**，而把：

```
P(Y = y | do(X = x)) - P(Y = y | do(X = x'))
```

称为**因果效应差（Causal Effect Difference）**。

此时，因果效应P(Y = y | do(X = x))等价于操纵图模型中的条件概率P_m(Y = y | do(X = x))。因此，计算因果效应的关键在于计算操纵概率P_m。

P_m与正常（无干预）条件下的概率P有两个相同的重要性：

1. **混杂变量的边缘概率不变**：P(Z = z)不随干预而变化，因为Z的取值不会由于去掉从Z到X的箭头而变化；
2. **条件概率不变**：P(Y = y | X = x, Z = z)不变，因为Y关于X和Z的函数f_Y = (Z, X, U_Y)并未改变。

因此，有如下等式：

- P_m(Y = y | X = x, Z = z) = P(Y = y | X = x, Z = z)
- P_m(Z = z) = P(Z = z)

在操纵图模型中X和Z是彼此独立的，即X的取值不会对Z取值造成任何影响，因此：

```
P_m(Z = z | X = x) = P_m(Z = z) = P(Z = z)
```

于是，将三个等式合在一起，即得因果效应：

```
P(Y = y | do(X = x)) = P_m(Y = y | X = x)
                     = Σ_z P_m(Y = y | X = x, Z = z) P_m(Z = z | X = x)
                     = Σ_z P_m(Y = y | X = x, Z = z) P_m(Z = z)
```

上述公式第一行由定义可得，第二行根据贝叶斯公式得到，第三行由X与Z在操纵图模型中的独立性得到。最后，干预X = x的因果效应可以完全根据无干预时的原始因果结构图进行计算：

**调整公式（Adjustment Formula）**：
```
P(Y = y | do(X = x)) = Σ_z P(Y = y | X = x, Z = z) P(Z = z)
```

对于Z的每一个取值z，该式计算X和Y的条件概率并取概率均值。上式右侧只包含无干预情况下的概率。

### 哪些变量可以放入调整公式中？

在进行干预时，由于要将X固定，并将所有指向X的箭头都去掉，即使得X的父节点失效，因此，应该将X的父节点放入调整公式中。令X的父节点集合为PA(X)，有如下性质：

**因果效应公式**：给定因果图，PA表示X的父节点集合，则X对Y的因果效应为：

```
P(Y = y | do(X = x)) = Σ_z P(Y = y | X = x, PA = z) P(PA = z)
```

其中z是PA的具体取值。

### 例13.3（辛普森悖论中用药与否的因果效应差计算）

继续例13.2。用X = 1表示用药，Y = 1表示患者恢复，Z = 1表示男性，Z = 0表示女性。通过干预"用药与否"，来计算用药在考虑性别因素的情况下对恢复情况的影响。

先固定X = 1（即用药），则有：

```
P(Y = 1 | do(X = 1))
= P(Y = 1 | X = 1, Z = 1) P(Z = 1) + P(Y = 1 | X = 1, Z = 0) P(Z = 0)
```

代入表13.1的数据：

- P(Z = 1) = (87 + 270) / (350 + 350) = 357/700
- P(Z = 0) = (263 + 80) / (350 + 350) = 343/700

```
P(Y = 1 | do(X = 1))
= 0.93 × (87+270)/(350+350) + 0.73 × (263+80)/(350+350)
= 0.93 × 357/700 + 0.73 × 343/700
= 0.832
```

类似地，将患者干预为未用药患者的因果效应为：

```
P(Y = 1 | do(X = 0))
= 0.87 × (87+270)/(350+350) + 0.69 × (263+80)/(350+350)
= 0.87 × 357/700 + 0.69 × 343/700
= 0.7818
```

**因果效应差**：
```
ACE = P(Y = 1 | do(X = 1)) - P(Y = 1 | do(X = 0))
    = 0.832 - 0.7818
    = 0.0502
```

所得结果表明用药患者的恢复率高于不用药患者的恢复率，即**该新药能帮助治愈患者**。这与原始数据中未用药恢复率（83%）高于用药恢复率（78%）的直观结论完全相反，充分展示了do算子调整混杂因素后得到正确因果推论的力量。

## 13.4 反事实推理

反事实推理作为因果推断的核心范式，旨在于构建"**如果... 则会...**"的逻辑框架，即在既定事实基础上推演潜在替代方案的可能结果。其本质是在平行时空假设下对两种情境进行比对分析：已实现的现实状态与未实现的假设状态。这种思维范式突破传统统计分析的局限，将因果关系的探索延伸至未被观测的潜在结果。

**例子**：某医疗团队为癌症患者制定治疗方案，在化疗（X = 1）与靶向治疗（X = 0）间选择实施后者。当观察到患者病情恶化时，主治医师反思："若采用化疗方案，生存期可能延长吗？"

该推理过程凸显反事实思维的关键认知跃迁：**决策时点的前瞻性预判与结果观测后的回溯性修正存在本质区别**。后者的修正源于对现实结果所揭示的新证据的整合，这种动态认知更新机制正是反事实推理区别于传统预测模型的核心特征。

### 子模型与反事实

13.2节介绍的子模型可用于表示假设变化的效应，即那些由反事实的前提所隐含的效应。

### 定义13.4（潜在响应）

给定结构因果模型M，令X和Y为V的任意两个变量子集。Y对行动do(X = x)的**潜在响应**表示为Y_x(u)，是方程组F_x中Y的解，即：

```
Y_x(u) = Y_{M_x}(u)                          (13.1)
```

模型M中的反事实Y_x(u)被定义为在被修改子模型M_x中Y的解。

方程（13.1）是因果推断中最基本的原则之一。它允许我们利用对现实的科学概念M，生成大量假设性问题的答案，如"如果X是x，那么Y会是什么？"同样的定义适用于当X和Y是变量集时，如果我们所说的M_x是一个模型，其中X的所有成员的方程都被常量替换。

### 定义13.5（反事实）

令X和Y为V的任意两个变量子集，反事实语句"在条件u下，如若X为x，则Y为y"，记作Y_x(u) = y，其中Y_x(u)为Y对X = x的潜在响应。

### 例13.4（反事实推理——教育拨款与学生素养差距）

2018年4月教育部印发了《教育信息化2.0行动计划》的通知，旨在提高农村中小学校智能设备覆盖率，缩小城乡中小学生数字素养差距。

令：
- X：国家年度教育信息化拨款额度
- H：农村学校智能设备覆盖率
- Y：城乡学生数字素养差距缩小幅度

其结构因果模型可以表示为M = ⟨U, V, F⟩，其中：

- U = {U_X, U_Y, U_H}
- V = {X, Y, H}
- F = {f_X, f_Y, f_H}

结构方程定义如下：

```
f_X: X = U_X
f_H: H = aX + U_H
f_Y: Y = bX + cH + U_Y
```

相应的因果图如**图13.3**所示：U_X → X，U_H → H，U_Y → Y；X → H（系数a），X → Y（系数b），H → Y（系数c）。

假设从统计数据中估计得a = 0.5，b = 0.7，c = 0.4。设有教育部对2022年的数据进行统计，发现三个数据分别为X = 0.5，H = 1，Y = 1.5。

**问题**：如果2022年的农村学校智能设备覆盖率多一倍，城乡学生素养差距的缩小量是多少呢？

### 反事实计算的三步骤

**步骤（a）——溯因（Abduction）**：利用现有的证据E = e，确定环境U。

在该因果模型中，变量的值由系数和外生变量决定，其中的外生变量将作用于所有个体。因此，可将教育部的统计数值X = 0.5，H = 1，Y = 1.5作为证据E，反推外生变量的值：

```
U_X = 0.5
U_H = 1 - 0.5 × 0.5 = 0.75
U_Y = 1.5 - 0.7 × 0.5 - 0.4 × 1 = 0.75
```

**步骤（b）——动作（Action）**：对模型M进行修改，移除等式中的变量并将其替换为X = x，得到修正模型M_x。

用H = 2表示2022年的农村学校智能设备覆盖率增加一倍，修改后的模型如**图13.4**所示：在图中，指向H的边被移除（即X不再影响H），H被固定为2。

**步骤（c）——预测（Prediction）**：利用修正模型M_x和环境U计算反事实Y_x(u)的值。

```
Y_{H=2}(U_X=0.5, U_H=0.75, U_Y=0.75)
= 0.5 × 0.7 + 2.0 × 0.4 + 0.75
= 0.35 + 0.80 + 0.75
= 1.9
```

即，若2022年的农村学校智能设备覆盖率增加一倍，则城乡学生素养差距的缩小量将从1.5变为1.9。

### 反事实计算总结

这一过程可解决任何确定性的反事实，即涉及单个实体的反事实，在这种情况下我们知道每个相关变量的值。结构方程模型能够回答这种性质的反事实查询，因为每个方程表示了变量值的确定机制。如果我们知道这些机制，也应该能够预测在某些机制发生变化时将获得何种值。因此，将反事实视为结构方程的派生属性是很合理的。

**三个步骤的逻辑**：
- 步骤（a）根据当前证据e解释过去（U）
- 步骤（b）使历史进程（最小程度上）扭曲，以符合假设前提X = x
- 步骤（c）基于对过去的新理解和新确立的条件X = x预测未来（Y）

9. **问答题**：在观察性研究中，为何随机对照实验（RCT）被认为是消除混杂因素的黄金标准？用因果图解释其原理。

   **解答**：在RCT中，被试被随机分配到实验组和对照组，使得处理变量（自变量）的取值完全随机化，不受任何其他变量的影响。在因果图上，这意味着所有指向处理变量的箭头都被切断，处理变量不再有任何父节点，因此混杂因素对处理变量的影响被消除。由于只有处理变量在变化而其他因素（包括混杂变量）保持不变，结果变量的任何变化都可以归因于处理变量。

10. **计算题**：以下是关于肾结石的数据集，其中肾结石的大小S ∈ {small, large}，治疗与否T ∈ {A, B}，以及康复与否R ∈ {0, 1}，其中R = 1表示康复。

    **表13.2：肾结石治疗方案与结石类型**

    | 结石类型（比例） | 治疗方案 A | 治疗方案 B |
    |-------------------|-----------|-----------|
    | 小结石 | 90/100 = 0.9 | 850/1000 = 0.85 |
    | 大结石 | 30/100 = 0.3 | 10/100 = 0.1 |
    | **合计** | 120/200 = 0.6 | 860/1100 = 0.782 |

    a) 请画出一个有向无环图来表达结石大小、治疗与否以及康复与否三者的关系。

    **解答**：结石大小S作为共同原因，同时影响治疗与否T和康复与否R。因果图：S → T，S → R，T → R。

    b) 计算P(R = 1 | do(T = A))和P(R = 1 | do(T = B))，并比较哪种方案治疗结果更佳。

    **解答**：
    使用调整公式，以结石大小S为调整变量：
    ```
    P(R = 1 | do(T = A)) = P(R = 1 | T = A, S = small) × P(S = small) + P(R = 1 | T = A, S = large) × P(S = large)
    ```
    - P(S = small) = (100 + 1000) / (200 + 1100) = 1100/1300
    - P(S = large) = (100 + 100) / (200 + 1100) = 200/1300
    ```
    P(R = 1 | do(T = A)) = 0.9 × 1100/1300 + 0.3 × 200/1300
                         = 990/1300 + 60/1300
                         = 1050/1300 ≈ 0.808

    P(R = 1 | do(T = B)) = 0.85 × 1100/1300 + 0.1 × 200/1300
                         = 935/1300 + 20/1300
                         = 955/1300 ≈ 0.735
    ```

    **结论**：治疗后的因果效应比较：P(R = 1 | do(T = A)) ≈ 0.808 > P(R = 1 | do(T = B)) ≈ 0.735，因此**治疗方案A更佳**。

    注意：这与原始数据中看到的总恢复率（A方案60% vs B方案78.2%）相反，这是一个典型的辛普森悖论案例——因为大结石患者更多地被分配到了A方案（100/200），而大结石患者的恢复率本身就更低。

11. **计算题**：党中央高度重视青少年的健康成长。自2011年农村义务教育学生营养改善计划启动实施以来，农村学生营养状况明显改善、身体素质明显提升、学习成绩显著提高。令X表示农村地区气候条件，H表示营养午餐计划，Y表示农村学生成绩。其结构因果模型可以表示为M = ⟨U, V, F⟩，其中U = {U_X, U_Y, U_H}，V = {X, Y, H}。F = {f_X, f_Y, f_H}定义如下：

    ```
    f_X: X = U_X
    f_H: H = aX + U_H
    f_Y: Y = bX + cH + U_Y
    ```

    相应的因果图如**图13.5**所示。假设根据统计数据估计得到a = 0.3，b = 0.5，c = 0.3。

    a) 根据国务院统计数据可得X = 0.2, H = 2, Y = 1作为证据E，请计算U中外生变量的值。

    **解答**：
    ```
    U_X = X = 0.2
    U_H = H - aX = 2 - 0.3 × 0.2 = 2 - 0.06 = 1.94
    U_Y = Y - bX - cH = 1 - 0.5 × 0.2 - 0.3 × 2 = 1 - 0.1 - 0.6 = 0.3
    ```

    b) 用H = 3表示更丰富的营养餐改善计划，请画出修改后的结构因果模型，并计算反事实数值Y_{H=3}。

    **解答**：
    反事实计算：
    ```
    Y_{H=3}(U_X=0.2, U_H=1.94, U_Y=0.3)
    = bX + cH_new + U_Y
    = 0.5 × 0.2 + 0.3 × 3 + 0.3
    = 0.1 + 0.9 + 0.3
    = 1.3
    ```

    修改后的结构因果模型：移除f_H，将H固定为3，即：
    - f_X: X = U_X（不变）
    - H = 3（固定）
    - f_Y: Y = bX + cH + U_Y（不变，但H使用新的固定值3）
