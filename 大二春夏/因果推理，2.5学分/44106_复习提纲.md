# 因果推理复习提纲

## 说明
- 本提纲以《复习 第一章 因果人工智能-绪论》《复习 第二章 因果推断》为主线。
- 其余课件主要补充了“因果赋能机器学习”“因果与大模型”“大模型赋能因果推断与发现”的应用型考点，适合单选、简答、开放题。
- 按考试题型整理：填空题、单选题、简答题、计算题、开放题。

## 一、最可能考的主干内容

### 1. 关联与因果
- 因果关系三个层级：关联、干预、反事实。
- 关联回答“看到什么”；干预回答“做了会怎样”；反事实回答“如果当时换一种做法会怎样”。
- 当前多数传统 AI 更擅长关联学习，难以直接回答干预和反事实问题。
- 关联的三种来源：真实因果、混杂偏差、选择偏差。
- 因果推理相对纯关联学习的价值：
  1. 提高可解释性。
  2. 支撑决策。
  3. 提升跨环境稳定性和 OOD 泛化。
  4. 处理公平与偏差问题。

### 2. 常见偏差与典型现象
- 混杂偏差：第三个变量同时影响处理变量和结果变量，造成虚假相关。
- 选择偏差：由于样本进入数据集的机制不随机，条件在选择变量上后产生虚假相关。
- 辛普森悖论：总体结论与分组结论相反，通常由混杂变量引起。
- 关键理解：
  1. 相关不等于因果。
  2. 调整不当可能“打开路径”，尤其是碰撞点。
  3. 不是所有变量都该控制。

### 3. 两大理论框架
- 潜在结果框架：用 \(Y(1)\)、\(Y(0)\) 描述同一个个体在不同处理下的结果。
- 结构因果模型/因果图模型：用 DAG、d-分离、do-算子刻画因果结构。
- 二者区别：
  1. 潜在结果更适合定义处理效应与估计问题。
  2. 因果图更适合表达变量关系、路径阻断、后门前门、干预分布。

## 二、填空题高频点

### 1. 基础概念
- 因果关系三个层级是：关联、干预、反事实。
- 个体因果效应 ITE：\(Y(1)-Y(0)\)。
- 平均因果效应 ATE：\(\mathbb{E}[Y(1)-Y(0)]\)。
- 条件平均因果效应 CATE：\(\mathbb{E}[Y(1)-Y(0)\mid X=x]\)。
- do-算子 \(do(T=t)\) 表示外部干预，把处理变量强制设为某个值。
- d-分离是图结构中的概念，条件独立是概率分布中的概念。
- Markov 条件：d-分离可推出条件独立。
- 忠实性条件：观察到的条件独立不是巧合，而由图结构决定。
- PC 算法属于基于约束的因果发现方法。
- FCI 算法用于存在潜在隐藏混杂时的因果发现。

### 2. 潜在结果框架假设
- SUTVA：一个个体的处理不会影响另一个个体的结果，同一处理无多个版本混淆。
- 可忽略性/无混淆：\((Y(1),Y(0)) \perp T \mid X\)。
- 重叠性/正值性：对任意协变量取值，接受处理与不接受处理的概率都大于 0。

### 3. 方法名称
- 倾向得分匹配：PSM。
- 倾向得分逆概率加权：IPW。
- 双重稳健方法：结果模型或倾向得分模型有一个正确即可保持一致性。
- 工具变量三大核心条件：相关性、排他性、无混杂性。

## 三、单选题高频点

### 1. 易混概念辨析
- “相关性强”不能推出“因果效应大”。
- 随机试验中，处理组和对照组均值差可直接估计 ATE；观测数据中通常不行。
- 碰撞点不能随便控制；控制碰撞点可能引入偏差。
- 后门变量要控制，工具变量通常不是拿来直接控制的，而是用于 IV 估计。
- PC 算法只能识别马尔可夫等价类，不能保证所有边方向都确定。
- CPDAG 中无向边表示方向无法仅由观测数据唯一识别。

### 2. 题目常考判断
- “FCI 比 PC 更适合存在隐藏混杂的情形”是对的。
- “倾向得分高维时更容易做原始协变量逐维精确匹配”是错的。
- “IPW 在倾向得分接近 0 或 1 时方差可能很大”是对的。
- “ATE 一定能准确反映个体最优决策”是错的，异质性处理效应会导致问题。
- “辛普森悖论的根源常是混杂变量分布不平衡”通常是对的。

## 四、简答题核心模板

### 1. 为什么需要从关联走向因果
- 关联学习能做预测，但难以解释“为什么”。
- 决策本质是干预问题，必须回答“如果采取行动会怎样”。
- 关联关系在环境变化下不稳定，因果机制更稳定。
- 公平、安全、医疗、推荐等高风险场景更需要因果分析。

### 2. 潜在结果框架的核心思想
- 同一个个体同时存在两个潜在结果：接受处理和不接受处理。
- 实际只能观测其中一个，另一个是反事实。
- 因果推断的困难本质上来自“反事实缺失”。
- 研究目标通常是从个体层面转向群体平均效应，如 ATE、CATE。

### 3. 为什么随机试验能识别因果效应
- 随机化使处理分配与潜在结果独立。
- 因此处理组和对照组在混杂变量上平均可比。
- 组间均值差就能解释为处理造成的结果差异，而非第三方因素导致。

### 4. 倾向得分方法的思想与局限
- 思想：先估计接受处理的概率 \(e(X)=P(T=1\mid X)\)，再利用它做匹配或加权。
- 目标：平衡处理组和对照组的协变量分布，使观测数据近似随机试验。
- 局限：
  1. 依赖可观测混杂假设。
  2. 倾向得分模型错设会影响结果。
  3. 倾向得分接近 0 或 1 时估计不稳定。

### 5. 工具变量法的思想与条件
- 适用场景：存在未观测混杂，无法仅靠控制协变量识别因果效应。
- 工具变量 \(Z\) 通过影响处理 \(T\) 间接影响结果 \(Y\)。
- 三个条件：
  1. 相关性：\(Z\) 与 \(T\) 相关。
  2. 排他性：\(Z\) 不直接影响 \(Y\)，只能通过 \(T\) 影响 \(Y\)。
  3. 无混杂性：\(Z\) 与影响 \(Y\) 的混杂因素独立。

### 6. PC 算法流程
- 从完全无向图开始。
- 逐步做无条件独立与条件独立检验，删去不应存在的边，得到骨架。
- 利用分离集识别 V-结构。
- 再用定向规则尽量确定边方向。
- 输出通常是 CPDAG，而不是唯一 DAG。

### 7. PC 算法局限
- 只能得到马尔可夫等价类。
- 对条件独立检验误差敏感。
- 忠实性假设可能被违反。
- 标准 PC 不能处理隐藏混杂，需用 FCI 等扩展。

## 五、计算题高频点

### 1. 直接算 ATE
- 常见给法：给出处理组平均结果和对照组平均结果。
- 随机试验中：ATE = 处理组均值 - 对照组均值。
- 题目可能问“若结果是成本/血压/损失，ATE 为负说明什么”：
  1. 若越小越好，则处理有效。
  2. 若越大越好，则处理不利。

### 2. 分层后算因果效应
- 先按混杂变量分层。
- 在每层内部计算“处理均值 - 对照均值”。
- 再按层样本占比加权平均。
- 常用来考辛普森悖论或“控制混杂后结论改变”。

### 3. 简单匹配思想题
- 给出若干样本的协变量 \(X\)、处理 \(T\)、结果 \(Y\)。
- 要求找协变量最相近的处理样本和对照样本。
- 再算匹配后的平均差值。
- 重点不在复杂计算，而在“匹配是为了让组间可比”。

### 4. 倾向得分/加权理解题
- 不一定要求复杂运算，常考“为什么要按倾向得分的逆加权”。
- 核心答法：给原本较少出现、较难被分配到当前处理的样本更大权重，从而恢复平衡后的伪总体。

### 5. 因果发现小题
- 给定一个简单图，判断哪些变量独立、给定谁后独立、哪里是 V-结构。
- 可能让你根据分离集判断边方向。
- 重点会放在链式、叉式、碰撞式三种基本结构。

## 六、开放题高频方向

### 1. 因果为什么能赋能机器学习
- 提高可解释性。
- 提升分布外泛化能力。
- 缓解数据偏差与虚假关联。
- 支持公平学习与稳健决策。

### 2. 因果与大模型结合的意义
- 大模型擅长语义理解和知识抽取，但本质上仍偏向关联建模。
- 因果方法可为大模型加入干预、反事实、去混杂、稳定推理能力。
- 结合方向：
  1. 用 LLM 处理文本混杂变量。
  2. 用 LLM 辅助找工具变量或构造因果图。
  3. 用反事实数据增强提升鲁棒性与公平性。
  4. 用因果模块改造注意力或推理流程。

### 3. 大模型赋能因果推断与发现
- 提示工程：基础提示、ICL、CoT、CausalCoT。
- 微调：提升特定因果任务性能，但成本更高。
- 融合传统方法：如 LLM+PSM、LLM+IV。
- 知识增强：上下文知识注入、知识图谱融合。

### 4. 推荐、医疗、法律等应用题怎么答
- 先指出存在混杂偏差、选择偏差或曝光偏差。
- 再说明纯相关分析会误导决策。
- 然后给出一个因果方案：
  1. 随机试验/A-B 测试。
  2. 倾向得分匹配或加权。
  3. 工具变量法。
  4. 反事实分析。
- 最后补一句：目标是提升可解释性、稳健性、公平性与决策可靠性。

## 七、考前速背清单

### 必背概念
- 关联、干预、反事实。
- 混杂偏差、选择偏差、辛普森悖论。
- ITE、ATE、CATE。
- SUTVA、可忽略性、重叠性。
- d-分离、Markov 条件、忠实性条件。
- 后门调整、do-算子、工具变量。
- PC、FCI、LiNGAM、ANM。
- PSM、IPW、双重稳健。

### 必背结论
- 随机试验最直接解决混杂问题。
- 观测数据做均值差通常不能直接当因果效应。
- 匹配和加权的目标都是平衡协变量分布。
- IV 解决的是未观测混杂问题，但工具变量条件很严格。
- PC 算法输出常是等价类，不一定能唯一确定方向。
- 大模型与因果结合是“语义理解 + 数值因果推理”。

### 必会举例
- 冰激凌销量与溺水人数：混杂变量是夏季高温。
- 鞋码与阅读能力：相关不代表干预有效。
- 推荐算法总体更优但分组后更差：辛普森悖论。
- 公鸡打鸣与太阳升起：不能把相关当因果。

## 八、答题建议
- 填空题：优先记英文缩写和标准定义。
- 单选题：注意“相关/因果”“观测/干预”“能否识别”这三组陷阱。
- 简答题：先给定义，再写机制，再写局限，最后给应用。
- 计算题：先判断是不是随机试验，再决定能不能直接算均值差。
- 开放题：一定把“混杂、选择偏差、反事实、可解释、稳健、公平”这些词写进去。
